# 社媒内容运营SOP

> 适用于 Facebook/Instagram、TikTok 内容运营，配合独立站 SEO + 广告做持续品牌曝光

---

## 一、内容发布节奏

### 平台分配

| 平台 | 发布频率 | 形式 |
|------|---------|------|
| **Facebook** | 每周3-5条 | 图文 / 短视频 / 链接 |
| **Instagram** | 每周3-5条 | 图文 post / Reels |
| **TikTok** | 每天1条 | 短视频（工厂/产品/科普） |
| **Pinterest** | 每周2-3条 | Pin 图 / Info Graphic |

### 每周内容日历模板

| 日期 | Facebook | Instagram | TikTok |
|------|----------|-----------|--------|
| 周一 | 工厂实拍图文 | 产品特写图 | 工厂 tour 视频 |
| 周二 | — | — | 技术科普视频 |
| 周三 | 行业资讯分享 | Reels（产品细节） | 客户反馈视频 |
| 周四 | 技术参数图 / 选型 Tips | 图文 post | 选型 Tips 视频 |
| 周五 | 客户案例 / 发货实拍 | Story（限时互动） | 发货/包装视频 |
| 周六 | — | — | 周末轻松向 |
| 周日 | — | — | — |

## 二、内容制作标准

### Facebook/Instagram 图文标准

| 元素 | 标准 |
|------|------|
| 图片尺寸 | 1080x1080px（Feed）/ 1080x1350px（Stories/Reels） |
| 视频时长 | 30秒-3分钟（Feed）/ 15-60秒（Reels） |
| 字幕 | 视频必须加字幕 |
| 文案长度 | 100-200字，自然描述，不硬广 |
| Hashtag | 3-5个相关标签（#LabelMaterial #BOPP #Packaging） |
| CTA | 引导："Link in bio" / "DM for quote" / "Comment 'INFO'" |

### TikTok 视频标准

| 参数 | 标准 |
|------|------|
| 尺寸 | 9:16 竖版（1080x1920）或 1:1（1080x1080） |
| 帧率 | 30fps |
| 时长 | 15-60秒（新手建议15秒） |
| 字幕 | 必须加字幕 |
| 文件格式 | MP4（H.264编码） |
| 封面 | 文字+产品图，高对比度 |

### Pinterest Pin 标准

| 参数 | 标准 |
|------|------|
| 图片尺寸 | 1000x1500px（长图）或 1000x1000px（方图） |
| 文件格式 | PNG 或 JPG |
| 描述 | 含关键词，200字以内，附CTA |
| Board 归属 | 归入对应分类 Board |

## 四、内容发布流程（标准5步）

### Step 1：内容规划（每周一）

- 根据当月营销节点规划本周内容主题
- 确认每个内容的发布平台和形式
- 在内容日历中填写计划

### Step 2：素材准备（周二/周三）

- 按发布计划拍摄或整理素材
- 按各平台标准调整尺寸/格式
- 制作文案草稿（含Hashtag）

### Step 3：内容制作（周三/周四）

- 编辑图片/视频（加字幕/品牌水印）
- 撰写发布文案（含CTA和链接）
- 制作各平台适配版本

### Step 4：排期发布（周四/周五）

| 平台 | 最佳发布时间（参考） |
|------|---------------------|
| Facebook | 周二-周四 9:00-11:00 / 14:00-16:00（当地时间） |
| Instagram | 周一-周五 11:00-13:00（当地时间） |
| TikTok | 12:00-14:00 / 18:00-20:00（当地时间） |
| Pinterest | 周六-周日 14:00-16:00（当地时间） |

- 使用 Meta Business Suite 预约 Facebook/Instagram 内容
- TikTok 建议手动发布（算法更偏好实时上传）

### Step 5：数据复盘（每周五）

| 指标 | Facebook/IG | TikTok | Pinterest |
|------|-------------|--------|-----------|
| 核心看 | Reach / Engagement | 播放量 / 互动率 | Saves / Clicks |
| 辅助看 | Click-through Rate | 粉丝增长 | Followers |
| 记录 | 在表格中记录每周数据 | 表格记录 | 表格记录 |

## 五、社媒与独立站的协同策略

### 内容引流路径

```
社媒内容（工厂实拍/技术科普）
    ↓
Link in Bio / Caption CTA
    ↓
独立站落地页（产品页/博客/About）
    ↓
询价表单转化
```

### 跨平台内容复用逻辑

| 原始内容 | 复用平台 |
|---------|---------|
| 工厂 tour 视频（TikTok） | Reels（IG）/ Facebook Feed |
| 材料对比图（Pin） | 博客文章配图 / Facebook 图文 |
| 技术科普视频（TikTok） | YouTube Shorts / LinkedIn |
| 客户案例（图文） | Instagram Carousel / Facebook 图文 |

### 链接策略

- 主页 Link：bit.ly 短链 → 独立站首页 + UTM 参数
- 帖子 CTA：引导到产品分类页 + UTM 参数
- Bio 链接：每个平台 Linktree 或 Bitly → 独立站 + UTM 参数

## 七、常见问题

| 问题 | 解决方案 |
|------|---------|
| 内容没互动 | 换发布时间 / 测试不同内容方向 / 加字幕 |
| TikTok 播放量低 | 加热门 BGM（但不要太娱乐化）/ 优化前3秒 |
| 询盘转化难 | 在帖子 CTA 加"Comment 'QUOTE'" / DM 引导 |
| 素材不够 | 每周固定安排工厂拍摄时间，积累素材库 |
