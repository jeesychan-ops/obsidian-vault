# B2B工厂独立站SEO整改SOP

> **适用场景**：B2B制造业独立站（模具/机械/汽配/定制加工等）已有基础但没询盘，预算有限/零预算，靠纯白帽SEO起量
> **预期周期**：3个月起量，6个月稳定出询盘
> **核心原则**：国内思维 → 海外思维 的全面翻译

---

## 阶段0：诊断（接手1-3天内完成）

### 诊断清单（4个致命问题）
- [ ] **机翻检测**：抽样10个产品页，看英文是否机器翻译（术语错误、句式生硬）
- [ ] **技术健康度**：用 Google Search Console + PageSpeed Insights 查收录率、速度、抓取错误
- [ ] **关键词错位**：列出网站当前主推词 vs 海外采购真实搜索词（用 Google Keyword Planner / 卖家精灵 / SEMrush 验证）
- [ ] **内容活跃度**：查最近6个月是否有更新（博客、案例、FAQ、工艺页）

> **快速判定**：4个问题命中2个以上 = 必须整改；命中3-4个 = 整改优先级最高

---

## 阶段1：英文文案全站重写（1-2周）

### 操作步骤
1. **删机翻**：把所有 `ctri/TR/DeepL` 一键翻译的页面标记出来
2. **找专业参考**：
   - 海外同行Top 5网站（用 Google 搜目标词，取前5个自然结果）
   - 行业白皮书 + 阿里国际站优质 listing（参考但不要抄）
3. **人工重写4类核心页**：
   - 首页：突出工厂实力 + 工艺专业度 + 定制能力
   - 产品工艺页：写清楚加工流程、参数、适用材料
   - 模具定制页：解释开模流程、起订量、报价逻辑
   - 工厂实力页：设备清单、产能、认证、案例
4. **B端文案口诀**：
   - 用海外买家能搜的词（custom / OEM / precision / small batch）
   - 写过程和逻辑（不是堆产品名）
   - 数字具体化（"500+模具/年"、"公差 ±0.02mm"）

### 验收标准
- 海外母语人士（或专业 freelancer）校对 0 错误
- 每个核心页 ≥ 800 字，关键词密度 1-2%
- 每个产品图加英文 ALT

---

## 阶段2：技术SEO全盘整改（1-2周，并行）

### 必做项
- [ ] **速度优化**：PageSpeed 移动端 ≥ 80 分
  - 压缩图片（WebP）、合并CSS/JS、CDN
- [ ] **抓取修复**：
  - robots.txt 正确
  - sitemap.xml 提交 GSC
  - 修复404、5xx、死链
- [ ] **结构清晰**：
  - 三层目录以内（首页 → 分类 → 产品）
  - URL 英文 + 连字符，无参数
  - 面包屑导航
- [ ] **Schema标记**：Product / Organization / BreadcrumbList / FAQ
- [ ] **Hreflang**：多语言站必备

### 工具
- Google Search Console（必装）
- PageSpeed Insights
- Screaming Frog（爬站）
- Rich Results Test

---

## 阶段3：海外真实关键词挖掘（持续）

### 关键词四象限
| 类型 | 例子 | 用途 |
|------|------|------|
| 产品核心词 | injection molding、plastic mold | 分类页主推 |
| 工艺词 | precision molding、insert molding | 工艺页主推 |
| 定制场景词 | custom plastic enclosure、OEM mold | 落地页/博客 |
| 长尾需求词 | small batch injection molding service | 博客/FAQ |

### 挖掘方法
1. **Google Suggest**：输入主词，记下拉框
2. **People Also Ask**：挖FAQ选题
3. **Google Keyword Planner**：看月搜索量 + 竞争度
4. **竞品反查**：用 SEMrush / Ahrefs 反查Top 3竞品关键词
5. **阿里国际站同行 listing**：参考"产品名 + 修饰词"组合

### 布局原则
- 标题 (Title)：1个主词 + 1个修饰词
- H1：与Title略有差异
- URL：/category/product-name/
- 正文：前100字出现主词
- 站内锚文本：用同义词/长尾词链接到相关产品

---

## 阶段4：持续内容更新（3个月+，核心动作）

### 更新频率建议
- **博客/工艺科普**：2-4篇/月（800-1500字）
- **FAQ**：1-2篇/月
- **案例/作品**：1-2个/月
- **行业新闻**：每周1次（可选）

### 内容选题来源
1. **客户真实问题**（销售/询盘邮件）
2. **People Also Ask 抓取**
3. **行业展会/新工艺**
4. **Google Trends 上升词**
5. **竞品博客选题**

### 内容类型模板
- **How-to**：如何选择模具钢材
- **Comparison**：注塑 vs 压塑 怎么选
- **Listicle**：模具定制 5 个常见误区
- **Case Study**：XX 客户定制案例解析
- **FAQ**：XX 工艺的 10 个问题

## 避坑提醒

1. ❌ **不要"建好就躺平"** — 谷歌把不更新的站判僵尸站
2. ❌ **不要机翻** — 谷歌和客户双输
3. ❌ **不要堆国内大词** — "塑胶模具" 没人搜
4. ❌ **不要追求颜值忽视技术** — 好看 ≠ 收录好
5. ✅ **白帽慢但安全** — 黑帽/采集/伪原创短期可能见效，但工厂站要长期主义
6. ✅ **B2B 站不要追流量，要追询盘** — 100 个精准B端访客 > 10000 个 C 端散客
