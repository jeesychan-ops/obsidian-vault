## 四、页面详细配置

### 4.1 首页（Homepage）模块顺序

```
[1] Hero Section（首屏）
    - 主标题：高质量不干胶大卷材料 — 源头工厂直供
    - 副标题：专注BOPP/PET/PP薄膜材料批发，支持定制规格
    - CTA按钮：Request a Quote（主要）| Browse Products（次要）
    - 背景：工厂实景视频或高质量工厂图片

[2] Trust Bar（信任条）
    - ISO 9001认证 | FSC认证 | 32年老厂 | 48小时报价响应
    - 图标+文字形式

[3] Core Product Categories（核心产品分类）3列网格
    - 薄膜材料（BOPP/PET/PP）
    - 纸类材料（热敏/铜版）
    - 特种材料（耐高温/冷冻）

[4] Why Choose Us（为什么选我们）4列
    - 工厂直供，无中间商
    - ISO认证，品质保证
    - 最低起订量灵活
    - 专业技术支持

[5] Product Highlights（产品亮点）滑动展示
    - 6-8个主打产品图+规格+询价按钮

[6] Application Industries（应用行业）6列图标
    - 食品饮料 | 医药 | 物流 | 化妆品 | 汽车 | 化工

    - "下载选型指南" | "查看材料对比表"

    - 3-4张工厂实景图+参观预约链接
```

## 五、技术SEO配置清单

### 5.1 每个页面必须配置的SEO字段

| 字段 | 说明 | 字符数建议 |
|------|------|-----------|
| **Meta Title** | 页面标题，含核心关键词 | 50-60字符 |
| **Meta Description** | 页面描述，含CTA | 150-160字符 |
| **URL Handle** | 简短、含关键词 | 如：/bopp-self-adhesive-roll |
| **Image ALT Text** | 每张图片描述性ALT | 含关键词，1-2句话 |
| **H1 Heading** | 页面主标题 | 含关键词，1个H1/页 |
| **Schema Markup** | 结构化数据（产品/FAQ） | JSON-LD代码 |

### 5.2 产品页Schema示例

```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "White BOPP Self Adhesive Label Roll",
  "description": "High quality white gloss BOPP self adhesive label material jumbo roll.",
  "sku": "BOPP-WG-500MM",
  "brand": { "@type": "Brand", "name": "[你的品牌名]" },
  "manufacturer": { "@type": "Organization", "name": "[你的公司名]" },
  "countryOfOrigin": "China"
}
```

### 5.3 FAQ页面Schema

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What is the minimum order quantity for BOPP label rolls?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Our standard MOQ for BOPP label rolls is 50 rolls per specification."
      }
    }
  ]
}
```

## 六、B2B询价工作流配置

### 6.1 客户分层策略

| 客户类型 | 特征 | 响应策略 |
|---------|------|---------|
| **新访客（无注册）** | 刚来网站，观望型 | 48小时内主动邮件跟进 |
| **表单询价客户** | 有明确需求 | 24小时内报价，附产品规格书 |
| **样品申请客户** | 验证品质阶段 | 优先处理，免费样品（小额）|
| **复购客户** | 已成交，有信任 | 加入VIP跟进名单 |
| **大客户（MOQ>500卷/月）** | 高价值 | 分配专属客户经理 |

### 6.2 邮件自动化序列

| 邮件 | 发送时间 | 内容 |
|------|---------|------|
| 询价确认 | 立即 | "感谢您的询价，我们已收到，将在24小时内给您报价。" |
| 正式报价 | 24小时内 | "您的询价已处理，这是我们的报价单（附件PDF），包含FOB价格和交期。" |
| 样品推荐 | 72小时后 | "如果您想先验证品质，我们推荐您申请样品包。" |
| 跟进 | 7天后 | "请问您对报价有任何问题吗？我们随时可以讨论。" |
| 价值内容 | 14天后 | "附送一份【如何选择合适的标签材料】选型指南，供您参考。" |
