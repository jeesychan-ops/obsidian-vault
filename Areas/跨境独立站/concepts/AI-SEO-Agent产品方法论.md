# AI SEO Agent 产品方法论

> 核心命题：把 founder 「高价值 / 低执行 / 流程长 / 讨厌做」的工作流，包装成订阅制 AI Agent。

## 1. 为什么这是好赛道

SEO 完美命中 founder 痛点四要素：
| 特征 | SEO 表现 |
|------|---------|
| 高价值 | 长期带来自然流量和品牌曝光 |
| 低执行 | 单点动作都不难，但流程碎 |
| 流程长 | 关键词→选题→写作→内链→CMS→复盘，跑通要几个月 |
| 讨厌做 | 反馈慢、琐碎、和"写代码改产品"抢时间 |

同类可切入的还有：邮件 outreach、客服 FAQ、目录站提交、社媒分发、Affiliate 渠道运营。

## 2. 商业模型验证

参考 [[raw/articles/2026-06-15-SEOBOT-月入6万美元案例|SEOBOT 公开数据]]：
- MRR $60K / 800+ 订阅 = ARPU ~$75/月
- LTV 高（SEO 是长跑，用户不会轻易换）
- 起步价 $49/月，对 founder 决策门槛低
- 退款承诺（首篇不满意退）→ 极大降低 AI 产品的不确定性顾虑

## 3. 定位差异：工具 vs Agent

| 维度 | AI Writer（工具） | SEO AI Agent |
|------|----------------|--------------|
| 解决 | 写一篇文章 | 跑完整个 SEO 流程 |
| 卖点 | 功能列表 | "少操心" / 增长结果 |
| 客户 | 内容运营 | 创始人 / 增长负责人 |
| 价格区间 | $9-29/月 | $49-199/月 |
| 续费动机 | 工具可换 | 工作流沉没成本高 |

→ 关键动作：**别挤写作工具货架，直接讲"我帮你跑完这件事"**。

## 4. MVP 切入口（适合个人独立开发者）

通用型拼的是综合能力（集成、运营、SEO 经验），**不适合普通人**。垂直 MVP 路径：

1. **绑定单一 CMS**：只服务 Shopify / Webflow / Framer 一家
2. **绑定单一内容源**：YouTube 链接 → 自动博客；产品页 → 自动 FAQ/Schema
3. **绑定单一垂直**：医美 / 法律 / 金融 / 母婴，垂直领域 fact-check 容易做壁垒
4. **绑定小语种市场**：英文 SEO 红海，西/葡/阿/日/德仍有空间
5. **绑定单一步骤**：只做"自动内链"或"自动 meta 优化"，做到极致

> 原则：**用户越容易理解，越容易付费**。

## 5. 官网转化设计 SOP

参考 SEOBOT 首页结构（按用户决策顺序排）：

1. **价值主张一句话** + 起价（49 美元起）
2. **结果数据**（impressions / clicks / 文章数 / 客户案例）
3. **功能矩阵**（覆盖范围打消顾虑）
4. **CMS 集成列表**（打消"接得上吗"）
5. **试用入口 + 退款承诺**（打消"会不会浪费钱"）
6. **FAQ**（打消"不会用"）
7. **信任背书**（PH 排名 / 媒体引用 / 真实截图）

## 6. 不可复制的优势 vs 可复制的优势

**不可复制（要避开别硬刚）**：
- 创始人 built-in-public 影响力
- 多项目 dogfooding 资源
- PH / Twitter 社区分发能力

**可复制（普通 founder 也能做）**：
- 定位成 Agent 而非工具
- 官网数据驱动 + 退款承诺
- 垂直 MVP + 单一 CMS 切入
- 把自己工作流产品化的思维方式

## 7. 风险与护城河

**风险**：
- 平台依赖（Google 规则变化）
- 内容质量（AI 内容被打击）
- 竞争（同质化严重）
- 预期管理（SEO 慢 vs AI 用户要快）

**护城河（不在"能生成文章"）**：
1. 真实数据反馈循环（哪些文章带来流量）
2. 多 CMS 集成工作流沉没成本
3. 垂直行业 know-how
4. 分发渠道
