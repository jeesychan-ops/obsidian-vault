# Shopify独立站建站方案

## 网站地图（Site Map）

```
www.[yourbrand].com
├── 首页（Home）
├── 关于我们（About Us）
├── 产品（Products）
│   ├── 薄膜材料（BOPP/PET/PP）
│   ├── 纸类材料（热敏/铜版）
│   ├── 特种材料（耐高温/冷冻/防伪）
│   └── 配件（热转印碳带）
├── 应用场景（Applications）
├── 资源中心（Resources）★ SEO核心
├── 询价（Request a Quote）★ 核心转化
├── 联系我们（Contact Us）
└── FAQ
```

## 首页模块顺序（10个模块）

1. Hero Section — 主标题 + CTA按钮
2. Trust Bar — ISO认证 / FSC认证 / 32年老厂
3. Core Product Categories — 3列产品分类
4. Why Choose Us — 工厂直供 / ISO认证 / 灵活MOQ / 技术支持
5. Product Highlights — 6-8主打产品
6. Application Industries — 6列行业图标
7. Technical Resources CTA — 资源下载引流
8. Factory/About Preview — 工厂实景
9. Testimonials — 客户评价
10. Footer — 全站链接

## 产品页模板

- 产品标题格式：`BOPP Self Adhesive Label Roll - White Gloss 500mm x 1000m`
- 关键参数面板（B2B核心）：材质/厚度/宽幅/长度/胶型/耐温/认证
- 询价表单（核心转化）：姓名/公司/邮箱/电话/数量/规格

## 技术SEO配置

| 字段 | 要求 |
|------|------|
| Meta Title | 50-60字符，含核心关键词 |
| Meta Description | 150-160字符，含CTA |
| URL Handle | 简短含关键词 |
| Image ALT | 含关键词，1-2句话 |
| Schema Markup | JSON-LD（产品/FAQ/Organization） |

## B2B询价工作流

| 客户类型 | 响应策略 |
|---------|---------|
| 新访客 | 48小时内主动邮件跟进 |
| 表单询价客户 | 24小时内报价 |
| 样品申请客户 | 优先处理 |
| 大客户（MOQ>500/月） | 专属客户经理 |

## 上线前检查清单

- Shopify后台：域名绑定+HTTPS生效
- 所有页面：Meta Title + Description 已填写
- 所有产品图：ALT文本已填写
- Google Search Console：网站已提交，站点地图已上传
- GA4：已安装，事件跟踪已配置
- 移动端测试：所有页面正常显示
- 页面速度：PageSpeed > 80分

## 月度维护任务

| 时间 | 任务 |
|------|------|
| 每周 | 检查询价表单，48小时内响应；发布1篇博客 |
| 每月 | 查看GA4数据；检查关键词排名 |
| 每季度 | 内容审计（死链/过期内容） |
| 每半年 | 竞品分析回顾 |
