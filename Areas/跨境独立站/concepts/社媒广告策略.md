# 社媒广告与内容策略

> 覆盖 Facebook/Instagram、TikTok、Pinterest 三大平台，配合独立站做品牌曝光 + 询盘引流

## 平台选择策略

| 平台 | 适用场景 | 内容类型 | B2B适配度 |
|------|---------|---------|-----------|
| **Facebook/Instagram** | 精准B2B受众再营销 + 行业社群 | 图文/短视频/Reels | ⭐⭐⭐⭐⭐ |
| **TikTok** | 工厂实拍/生产过程/品牌认知 | 15-60秒短视频 | ⭐⭐⭐ |
| **Pinterest** | 视觉类产品（标签材料/包装） | Pin图/Info Graphic | ⭐⭐⭐ |

## Facebook/Instagram 广告

### B2B受众定向

| 受众类型 | 设置方式 | 用途 |
|---------|---------|------|
| **核心受众** | 兴趣词：label material, packaging, printing | 冷启动 |
| **再营销受众** | 网站访客（Pixel）/ 询价未转化用户 | 二次触达 |
| **相似受众** | Lookalike 1% 基于询价客户 | 拓量 |
| **行业定向** | 公司规模50-999人 / 制造业/包装行业 | 精准B2B |

### 广告素材方向（B2B工业品）

| 内容方向 | 示例 | 效果 |
|---------|------|------|
| 工厂实拍视频 | 生产线运转 / 产品打包 / 质检过程 | 信任感 |
| 产品细节特写 | 材质纹理 / 印刷效果 / 卷芯特写 | 专业感 |
| 技术参数图 | 材料对比表格 / 规格表截图 | 采购参考 |
| 客户案例 | "XX客户月用量5000卷，合作2年" | 背书 |
| 选型指南类 | How to Choose infographic | 吸引询盘 |

### 投放策略

| 阶段 | 目标 | 预算占比 |
|------|------|---------|
| 冷启动 | 加购/询价事件积累 | Pixel安装优先 |
| 数据积累后 | 再营销 + Lookalike | 70%再营销 |
| 规模期 | 相似受众拓量 | 50%相似受众 |

### Instagram Reels 策略

- **时长**：15-30秒（快节奏）
- **字幕**：必须加字幕（85%无声观看）
- **音乐**：轻商业感音乐，不要太娱乐化
- **CTA**：Link in bio 引导到询价页
- **频率**：每天1条或每周5条

## TikTok 内容策略

### B2B工厂 TikTok 内容方向

| 内容类型 | 示例 | 目的 |
|---------|------|------|
| 工厂 tour | "Take you inside our label material factory" | 信任/真实 |
| 生产过程 | 线切割 / 涂布工序 / 质检过程 | 专业感 |
| 产品特写 | BOPP卷展开 / 印刷效果对比 | 产品力 |
| 技术科普 | "What is BOPP film and how is it made?" | GEO/搜索 |
| 客户发货 | 实拍装柜 / 客户反馈截图 | 背书 |
| 选型 Tips | "3 things to check before ordering label rolls" | 引流 |

### TikTok 导流独立站路径

```
TikTok视频 → Link in Bio（bit.ly短链）→ 独立站首页或产品页
             ↓
        加CTA文字："Tap the link to request a quote"
```

> **注意**：TikTok 适合品牌认知，不适合直接B2B询盘转化，核心目标是引流到独立站

### 内容日历

| 日期 | 内容方向 |
|------|---------|
| 周一 | 工厂实拍 / Behind the scenes |
| 周三 | 产品特写 / 技术科普 |
| 周五 | 客户案例 / 发货实拍 |

## Pinterest 策略

### 适用场景

- 视觉类产品：标签材料 / 包装设计 / 品牌视觉
- 目标：设计师/包装工程师/品牌方

### Pin 图内容类型

| 内容 | 示例 | 落地页 |
|------|------|--------|
| 材料对比图 | PET vs PP vs BOPP visual comparison | 博客文章 |
| 应用场景图 | Food packaging label application | 产品分类页 |
| 选型流程图 | How to choose label material flowchart | 博客文章 |
| 工厂实拍图 | Production facility overview | About页面 |

### 优化要点

- 每张 Pin 图加详细描述，含关键词
- Board 分类：Label Materials / Packaging Solutions / Factory Tour
- 链接指向相关博客或产品页

## 社媒广告 vs Google Ads

| 对比维度 | Google Ads | 社媒广告 |
|---------|-----------|---------|
| 用户意图 | 强（主动搜索） | 弱（被动刷到） |
| 转化路径 | 短（搜索→点击→询价） | 长（内容→认知→信任→询价） |
| 适合产品 | 有明确需求的产品词/供应商词 | 无明确需求但有行业相关性 |
| 优化难度 | 中（数据明确） | 高（品牌效果难量化） |
| 预算门槛 | $1000+/月开始有效 | $500+/月开始有效 |

> **建议**：Google Ads 优先（强意图），社媒广告作为辅助（品牌 + 再营销）
