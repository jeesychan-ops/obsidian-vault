# SEO内容策略

## 站内SEO

| 字段 | 优化要求 |
|------|---------|
| Title/Description | 每个页面独立，含关键词，字符合规 |
| URL Handle | 简短、含关键词、连字符分隔 |
| H1 Heading | 每页1个，含核心关键词 |
| Image ALT | 描述性，含关键词 |
| 内链 | 相关页面互相链接，形成网状结构 |
| Schema Markup | JSON-LD（产品/FAQ/Organization） |

## 内容策略

| 类型 | 频率 | SEO价值 |
|------|------|---------|
| 选型指南（Selection Guide） | 每2周1篇 | 长尾词+GEO |
| 材料对比（Material Comparison） | 每月1篇 | 竞品词 |
| 应用行业指南 | 每月2篇 | 行业流量 |
| 技术百科 | 每月1篇 | 定义型词 |

## 技术SEO

- 页面速度：PageSpeed Insights > 80分
- 移动端适配：Google Mobile-Friendly Test 通过
- HTTPS：已启用
- 结构化数据：产品页/FAQ页/Organization
