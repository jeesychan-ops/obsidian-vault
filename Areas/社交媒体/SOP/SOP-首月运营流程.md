# 首月运营SOP

## 第1天：账号注册

- [ ] 一卡一机一号，准备好实名认证的手机号
- [ ] 头像：用本人真实照片
- [ ] 昵称：带行业关键词，不留联系方式
- [ ] 简介：自然描述，不堆砌关键词
- [ ] 绑定手机号 + 实名认证

## 第2-6天：养号

- [ ] **不要发布任何笔记**
- [ ] 每天浏览30-60分钟，搜索赛道关键词
- [ ] 认真看完相关笔记，点赞+收藏+用心评论（3-5句话）
- [ ] 关注10-20个同赛道博主
- [ ] 每天互动20-30条，间隔随机，模拟真人

## 第7-10天：发布首篇

- [ ] 选择"反向安利+真实体验"角度
- [ ] 标题制造信息差或情感共鸣
- [ ] 正文：痛点引入 → 个人经历 → 干货输出 → 自然引导关注
- [ ] 审核：无违禁词、无联系方式、图片清晰

## 第1-4周：持续输出

- [ ] 每周3-4篇，按选题矩阵执行
- [ ] 黄金时段发布：7:30-9:00 / 12:00-13:00 / 19:00-21:00
- [ ] 评论2小时内回复
- [ ] 每天检查数据：曝光、点击率、互动率

## 每周复盘

- [ ] 查看笔记曝光量趋势
- [ ] 分析CTR和互动率
- [ ] 找到表现最好的内容方向
- [ ] 调整下周选题计划
