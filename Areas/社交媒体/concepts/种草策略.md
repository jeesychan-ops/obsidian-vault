# 种草策略

## KOL/KOC种草选择

### KOL（头部达人）
- 粉丝量：10万+
- 适合品牌背书、扩大知名度
- 成本高，适合大促节点

### KOC（素人博主）
- 粉丝量：1000-10万
- 真实感强，适合口碑种草
- 成本低，适合长期铺量

## 种草内容结构

### AESTHETIC公式（种草文案）
- **A**uthentic：真实体验感
- **E**motional：情感共鸣
- **S**pecific：具体细节（数字、场景）
- **T**rusted：信任背书
- **H**onest：诚实（先说缺点再夸优点）
- **I**ncentive：行动激励

### 种草笔记结构
```
【开头】场景代入（什么情况下我买了/用了）
【正文】
  1. 真实体验过程
  2. 具体感受（好在哪/有什么不足）
  3. 适合人群说明
  4. 横向对比（如有）
【结尾】总结+购买建议
```

## 选品种草要点

- 优先选有真实使用场景的产品
- 强调"自用"而非"广告"
- 包含价格信息和购买渠道（站内官方链接）
- 真实缺点要说，增加可信度
