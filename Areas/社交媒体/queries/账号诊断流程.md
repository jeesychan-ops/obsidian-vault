# 账号诊断流程

## 诊断维度

### 1. 数据层面（先定量分析）

| 指标 | 参考标准 | 异常信号 |
|------|---------|---------|
| 曝光量 | 新号500+/篇 | <200持续 → 账号问题 |
| 点击率 | 5-10% | <3% → 封面/标题问题 |
| 互动率 | 3%+ | <1% → 内容质量问题 |
| 涨粉数 | 互动人数的10-20% | 互动高但不涨粉 → 引导关注不足 |

### 2. 内容层面（定性分析）

- 选题是否贴合目标人群需求
- 封面是否清晰、视觉有冲击力
- 标题是否制造信息差或情感共鸣
- 正文结构是否清晰（痛点→干货→引导）
- 是否有违禁词/联系方式

### 3. 账号层面

- 养号期是否执行到位
- 是否有违规记录
- 发布时间是否在黄金时段
- 评论互动是否及时

### 4. 赛道层面

- 赛道竞争激烈程度
- 内容是否与竞品差异化
- 是否有季节性/时效性红利
