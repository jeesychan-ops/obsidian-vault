## 二、有价值·需验证策略

### ⚠️ 哑铃策略（借鉴）

**核心逻辑**: 50%现金流资产（低波动高分红）+ 50%科技行业（高成长高波动），不投中间平庸板块

| 指标 | 数据 |
|------|------|
| 回测周期 | 2013年1月-2026年1月（13年） |
| 累计收益 | 495.08% |
| 年化收益 | 14.62% |
| 正收益年份 | 13年中10年正收益 |
| 最大回撤 | 50.34%（2016年1月熔断） |

**复制方法**:
- 现金流类: 自由现金流/中证红利/沪深300低波动
- 科技类: 申万电子/半导体/芯片

> ⚠️ **借鉴**: 最大回撤50%对9.4万资金心理压力大。建议用ETF替代，定投缓解回撤。

### ⚠️ 跟着"赚钱"公司买

| 属性 | 详情 |
|------|------|
| 回测周期 | 2024年8月-2025年8月（1年牛市） |
| 累计收益 | 150%+ |
| 最大回撤 | 9% |
| 筛选条件 | 中性ROE>15%，近1年ROE>5%，20日股价偏离度<8%，真实盈利排序 |

**公式**:
- 真实盈利 = TTM(经营现金流量净额) / TTM(归属于母公司所有者的净利润)
- 中性ROE = TTM(归属于母公司所有者的净利润) / AvgQ(归属母公司股东权益合计, 4, 1)

> ⚠️ **借鉴**: 回测期仅1年（牛市），150%+收益不可持续，过拟合嫌疑大。但'真实盈利'和'ROE过滤'的选股思路有价值。
