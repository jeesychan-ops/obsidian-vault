# 板块轮动方法论

> 板块轮动的本质：市场心理周期 × 行业周期与政策 × 资金预期差博弈 = 物极必反

## 一、轮动类型与结构

### 板块层级
| 类型 | 特征 | 操作 | 持续性 |
|------|------|------|--------|
| **主线** | 建制完整（龙头+中军+先锋+补涨） | 重仓参与 | 最长 |
| **次主线** | 结构略弱，双主线交替 | 中等仓位 | 中 |
| **支线** | 板块效应不强，助攻属性 | 轻仓试错 | 短 |
| **一日游** | 消息刺激+一字板，龙头需观察 | 不追 | 极短 |

### 板块结构完整性判断
- ✅ **建制完整**：龙头（连板）+ 中军（大成交额）+ 先锋（日内进攻）+ 补涨（资金后期选择）+ 套利品种
- ✅ **强板块**：涨停数量多 + 内部结构紧密（龙头涨停后龙2几分钟内跟随）+ 龙头稳定不换
- ❌ **弱板块**：结构残缺（只有龙头没跟风）→ 一两天结束
- ❌ **灭亡信号**：出现抢龙头（多只争夺龙头）→ 接近尾声
- 🔴 **退潮终极信号**：板块最冷门股票开始上涨 → 清仓离场

## 二、强度量化指标

### 硬指标（可量化）
1. 竞价涨幅前列 + 开盘半小时 3 只以上底部放量上攻
2. 早盘一字板数量（越多越强）
3. 涨停家数：一般不少于 10 家才能形成主流效应
4. 百万手大单封死 = 给板块打空间
5. 板块内大盘蓝筹股异动往往是轮动先兆
6. 板块内有不断趋势新高的股票 = 大资金已潜伏，持续性强

### 大盘环境支持
- 成交量 < 8000亿 → 板块难以持续
- 成交量 > 1万亿 → 容易出主线
- 指数在 5 日线上方 → 容易赚钱
- 指数在 5 日线下方 → 容易亏钱

## 三、7 种不能做的场景（操作红线）

| # | 场景 | 原因 | 正确做法 |
|---|------|------|----------|
| 1 | 板块高潮 | 次日必分歧 | 放弃接力，尤其警惕下午被动涨停的"弱转强" |
| 2 | 分歧后回流 | 别人拿先手你接盘 | 回流不加强就空仓 |
| 3 | 高位龙头 | 博弈价值低 | 没有新故事不接力 |
| 4 | 连续缩量加速 | 危险信号 | 不追，等分歧 |
| 5 | 龙头还强提前切补涨 | 节奏错误 | 龙头强时仍做龙头 |
| 6 | 高潮次日接力 | 接盘 | 识别分歧彻底程度，不急进场 |
| 7 | 连续跌停博反弹 | 飞刀 | 切忌，等企稳 |

## 四、轮动信号识别

### 时间维度
- **见顶→恐慌→惯性下跌→情绪修复**：第4-5天可关注
- **二轮上涨**：力度减弱、速度加快 → 后期谨慎

### 传导路径
| 传导类型 | 路径 |
|---------|------|
| 产业上下游 | 下游产品→中游加工→上游原材料 |
| 政策导向 | 国家政策→相关行业板块 |
| 突发消息 | 利好刺激→板块崛起 |
| 比价效应 | 龙头涨→同属性补涨 |

### 板块间关系
- **助攻**：相似板块跟风但未盖过原板块 = 秩序良好 → 可参与
- **拆台（跷跷板）**：大蓝筹与中小创互相制约 → 谨慎/空仓
- **协同作战**：多板块同涨 → 可积极参与

## 五、操作框架

### 选股逻辑
1. 在热点板块中按自己模式选股
2. 买入灵魂股或联动性最强的股票
3. 介入时机：板块集体爆发前为最佳
4. 大成交额个股 = 大资金青睐 = 持续性强

### 持仓管理
- 介入板块龙头：只要龙头不断板就持有，断板即卖
- 老龙头滞涨断板 → 切换补涨龙
- 从中高位切换到首板低位（高低切）
- 没有机会 → 空仓等待

### 每日复盘流程
1. 复盘近期异动板块，归类到自定义板块
2. 关注消息面（财联社等），判断利好程度
3. 预判哪些板块可能异动
4. 判断：哪个板块最强？板块风格（题材 vs 赛道）？
5. 判断板块强度（竞价一字数量、前排封单、梯队完整度）
6. 午盘后观察是否补量或回落
