# 仓位管理原则

> 仓位管理的执行细节。三层仓位结构总览见 [[投资理念总纲]]。

## 三层仓位结构

```
总仓位 ~94,889元
├── 防御层 50~55%   → 长电+移动+石油（稳定收息，股息3~5%）
├── 困境反转层 20~25% → 分众+海大+紫金（等估值修复，不加仓）
├── 进攻层 25~30%    → 双低可转债（~24,000元）
└── 子弹仓 5~10%    → 预留现金（~5,000~10,000元）
```

## 建仓节奏

| 阶段 | 仓位 | 说明 |
|------|------|------|
| 试探仓 | 5% | 验证判断，确认逻辑 |
| 确认仓 | 10-15% | 方向确认后加仓 |
| 重仓上限 | 30% | 任何标的最高不超过30% |

## 止损体系

### 软止损（组合层面）

- 整体回撤 -10% → 警戒线
- 整体回撤 -15% → 降仓信号
- 整体回撤 -20% → 强制减仓

### 硬止损（个股层面）

- 单只浮亏超过 **25%** → 必须重新评估持仓逻辑
- 双低可转债 **-3%** → 硬止损（债底保护，执行要坚决）

### 止盈

| 策略 | 止盈条件 |
|------|----------|
| 双低可转债 | +5% 卖一半；强势回撤 -1% 清仓 |
| 股票仓位 | +20% 部分止盈 |
| 行业指数 | 3日跌超8%集体止盈 |

## 行业分散原则

- 同一行业个股合计不超过总仓位 **30%**
- 永远留子弹（子弹仓 5~10%），不在低谷时被迫割肉

## 与其他概念的关系

- 三层结构是 [[投资理念总纲]] 资产配置的具体化
- 进攻层 = [[双低可转债策略]]
- 双低可转债的 -3% 止损止盈是本原则的具体执行
