# 每周复盘 SOP

## 复盘内容

1. **持仓健康度检查** — 运行 `stock_analysis.py`，浮亏扩大？浮盈守住？
2. **双低池子更新** — 运行 `cb_dual_low_scan.py`，有无新机会？持仓债调仓？
3. **市场环境判断** — 大盘情绪、可转债市场整体中枢、热点方向
4. **策略有效性评估** — 本周收益 vs 沪深300，双低 vs 大盘
5. **下周操作计划** — 明确 1~3 个具体操作，不开新仓

## 输出模板

```
📅 每周复盘 · YYYY-MM-DD

📊 本周表现
- 账户总值：X元（上周Y元，±Z%）
- 持仓涨跌：XXXX
- 双低进攻仓：XXXX

🎯 下周计划
1. [操作] 具体标的 + 条件
2. [操作] 具体标的 + 条件
3. [操作] 具体标的 + 条件（可选）

💡 市场观察
- 大盘：XXXX
- 可转债：XXXX
- 关注机会：XXXX
```

## 关键检查项

### 持仓健康度（stock_analysis.py）
- 浮亏是否扩大？
- 浮盈是否守住？
- 有无单只超过 25% 浮亏（硬止损线）？

### 双低池子（cb_dual_low_scan.py）
- 有无新的双低机会（价格<110，溢价率<15%）？
- 持仓可转债是否触及止盈/止损线？
