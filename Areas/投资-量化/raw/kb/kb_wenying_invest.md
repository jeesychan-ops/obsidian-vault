## 二、投资红线（绝对不能碰）

1. ❌ 不预测大盘短期涨跌
2. ❌ 不追涨停板（流动性陷阱）
3. ❌ 不 ALL IN 单只股票
4. ❌ 不碰自己不懂的品种
5. ❌ 单只仓位不超过总仓位 **20%**（特殊情况不超过 30%）

## 三、核心操作原则

| 类型 | 规则 |
|------|------|
| **仓位管理** | 试探仓 5% → 确认仓 10-15% → 重仓不超过 30% |
| **软止损** | 组合整体 -10%、-15%、-20% 设警戒线 |
| **硬止损** | 单只浮亏超过 25% 必须重新评估 |
| **分散** | 同行业个股不超过 30%，永远留子弹 |
| **盈亏比** | 买前先算清楚，盈亏比 > 1.5:1 才动手 |

## 六、核心策略：可转债双低策略 ⭐

### 筛选条件

| 指标 | 条件 |
|------|------|
| 价格 | 100~115 元 |
| 溢价率 | < 20% |
| 评级 | AA 级以上 |
| 到期收益率 | > 0% |
| 剩余年限 | > 1 年 |
| 排除 | 已公告强赎、正股 ST/*ST |

### 双低值计算
> **双低值 = 价格 + 溢价率 × 100**，越小越好

### 操作规则

| 规则 | 参数 |
|------|------|
| 最大持仓 | 3~5 只，单只 ≤ 15% |
| 买点 | 回踩不崩再买，不追高 |
| 止损 | -3% 硬止损 |
| 止盈 | +5% 卖一半，强势回撤 -1% 走 |
| 持有周期 | 1~4 周为主 |
| 调仓频率 | 每月检视一次 |

### 预期收益
- **年化**：约 15~25%（强势市场）
- **最大回撤**：< 5%（债底保护）
- **夏普比率**：> 1.0

## 九、微信文章分析 SOP

### Step 1：抓取
```bash
python3 /root/.hermes/scripts/wechat_fetch.py --url <链接>
```
保存至：`articles/YYYYMMDD_标题.json`

### Step 2：可行性评估（6条）
1. 逻辑是否成立 — 因子有没有经济学/行为金融学依据
2. 回测数据是否可信 — 年化>30%需警惕幸存者偏差/过拟合
3. 数据能否获取 — 散户能拿到所需数据吗
4. 资金量能否执行 — 9.4万够不够分散
5. 执行成本 — 手续费、滑点、时间精力
6. 是否符合用户约束 — 不止损、调仓换仓为主

### Step 3：评分与结论
- 评分维度：逻辑完整性 / 数据可获取性 / 落地可行性 / 互补性
- 结论：**✅采纳** / **⚠️借鉴** / **❌不适用**

### Step 4：归档
- 采纳 → 存入 `strategies/`
- 借鉴 → 提取因子，融入现有策略
- 不适用 → 记录原因
