# 《可转债投资魔法书第二版》—— 结构化工笔记

> 本笔记聚焦书中**超出已有双低策略筛选条件之外**的核心理论、历史回测数据、极端市场表现及安道全独创仓位管理方法。已有双低筛选条件（价格+溢价率等）不重复，重点补充策略的理论根基与实战经验。

---

## 一、核心理念：可转债的底层逻辑

### 1.1 为什么"100%保底，上不封顶"？

- **保底来源**：可转债本质是"债"——合同规定到期还本付息。即使股价跌到脚后跟，上市公司必须还钱（法律+合同+资质三重保障）。
- **上不封顶来源**：可转债可转换为股票（期权属性）。转股价一般在面值附近，当股价超越转股价时，转债价格随之上涨。
- **历史铁证**：可转债历史上到期保本的概率 **100%是100%**。

### 1.2 "100元以下买入，等130强制赎回"的操作纪律

这是全书的**根本操作纲领**：

- **100元以下** → 越跌越买（跌破面值 = 折价买债，安全边际最大）
- **100~130元之间** → 不动如山（耐心持有，等待上市公司"打工"拉抬股价）
- **130元以上** → 视情况卖出或继续持有（触发强制赎回条件，胜利大撤退）

> 可放宽标准：接近100元、或回售保护价以下、或到期收益率为正的价位也可买入。

### 1.3 为什么上市公司"必然"拉抬股价到130%以上？

可转债发行公司的终极目标：**把债主变成股东（不用还钱）**。操作路径：
1. 股价低迷 → 不愿转股（债主持债观望）
2. 上市公司着急 → 动用"下调转股价"杀手锏
3. 配合利好/业绩 → 拉抬股价到转股价 **130%以上**
4. 满足条件 → 触发"强制赎回" → 持有人被迫转股 → 上市公司完成债转股 → **双赢**

**核心洞察**：保持股价在转股价130%以上一段时间，**对上市公司最有利**。换言之，持有可转债获得30%以上收益，是大概率事件。

## 二、可转债四大要素（浓缩理解）

### 2.1 转股价 —— "可转债的心脏"

- **定义**：可转债转换为股票时的约定价格
- **确定方式**：募集说明书公告日前20日均价和前一日均价两者中的较高者
- **动态调整**：遇分红、送股、增发等自动调整（按公式）
- **核心判断**：转股价 < 当前股价 → 转债有价值（价内）→ 价格上升；转股价 > 当前股价 → 无转股价值 → 价格向面值靠拢

### 2.2 向下修正转股价条款 —— "上市公司的起重机"

- **触发条件**：股价在Ｎ个交易日内低于转股价某个阀值（如连续20日中10日低于转股价85%）
- **性质**：上市公司"**有权**"（非义务）下调转股价，需股东大会2/3通过
- **关键限制**：修正后转股价 ≥ 每股净资产、≥ 股票面值、≥ 前20日均价与前一交易日均价的较高者
- **实战意义**：PB < 1的公司下调空间极有限（受每股净资产限制）；下调转股价立竿见影地拉抬转债价格

### 2.3 强制赎回条款 —— "胜利的号角 / 玻璃天花板"

- **触发条件**：股价在Ｎ个交易日内维持在转股价 **130%** 以上（常见：30日中至少20日 ≥ 130%）
- **后果**：上市公司有权以103元左右赎回全部未转股的可转债
- **本质**：逼迫持有人"二选一"——要么转股获30%+收益，要么103元被低价赎回
- **性质**：上市公司"**有权**"，可直接执行（无需股东大会）
- **五星红旗**：强制赎回公告 = 胜利大撤退的号角

### 2.4 回售条款 —— "持有人的保护伞"

- **触发条件**：股价在Ｎ个交易日内低于转股价某个阀值（如最后2个计息年度，连续30日低于转股价70%）
- **后果**：**持有人有权**按面值103%左右将转债卖回给公司
- **性质**：持有人"**有权**"，是对上市公司的反制武器
- **实战意义**：回售是制约上市公司"赖账不下调转股价"的杀手锏。面临回售压力，上市公司通常会选择拉抬股价或下调转股价

### 四大要素权力对照

| 条款 | 权力归属 | 性质 |
|------|---------|------|
| 转股价 | 合同约定 | 动态调整 |
| 下调转股价 | 上市公司有权 | 需股东大会批准 |
| 强制赎回 | 上市公司有权 | 可直接执行 |
| 回售 | **持有人**有权 | 对上市公司的反制 |

- 2008年10月最低跌至 **82.7元**（市场传言母公司南山铝业即将破产）

- 此后随阶段性牛市最高涨至 **199元**（不到一年，收益率 **140%**）

- 仅一个月后涨至 **102.59元**（收益 **19.36%**）
- 到期收益率（假设）为保守估计 **40%+**

### 4.3 "面值-高价折扣法"在极端行情下的回测验证

> 安道全团队明确表示："面值-高价折扣法"原理，经过 2008~2013 年市场的洗礼，经受住了时间的考验，证明是安全且行之有效的。

## 五、安道全独创仓位管理方法："三条安全线"

### 5.1 为什么需要三条安全线？（解决四大痛点）

| 痛点 | 解决方案 |
|------|---------|
| 有的转债几年徘徊在面值以下，时间成本高 | 下调三条线（提高买入门槛） |
| 有的转债从未跌破面值，一飞冲天 | 上调三条线（放宽买入条件） |
| 同时多只转债，选谁不选谁 | 看谁先满足三条线，先到先得 |
| 买多少？什么节奏？ | 三条线对应三种建仓比例 |

### 5.2 三条安全线的定义

**三条线 = 三个安全价格，由安全到更安全排列：**

| 安全线 | 名称 | 定义 | 含义 |
|-------|------|------|------|
| **第一条** | **安全建仓线** | **到期安全价**（含税） | 到期还本付息的总价值，买入即保本 |
| **第二条** | **安全加仓线** | **当期回售价** | 触发回售的价格，有回售保护 |
| **第三条** | **安全重仓线** | **面值100元**（或正年化价格） | 跌破面值，安全边际最大 |

**到期安全价计算公式**（安道全独创算法）：
```
到期安全价 = 到期赎回价(含最后1年利息) + (除最后1年外所有未付利息之和) × 80%
```

- 例：某转债6年利息0.5%、0.5%、1%、1%、2%、2%，到期赎回价106元（含最后一年2%）
- 到期安全价 = 106 + (0.5+0.5+1+1+2) × 80% = **110元**

### 5.3 仓位控制三大比例

#### (1) 单只转债内建仓比例（建仓线:加仓线:重仓线）

| 风格 | 比例 | 说明 |
|------|------|------|
| 平衡型（推荐） | **3:4:3** | 最有普遍性，适合新手起步 |
| 积极建仓型 | 5:3:2 | 偏向尽早建仓 |
| 重仓等待型 | 2:3:5 | 偏向在低价重仓 |
| 稳健型 | 3:3:4 | 均衡偏保守 |

> 例如3:4:3意味着：满足建仓线最多买30%，满足加仓线再加到60%，满足重仓线才加满到100%。

#### (2) 单只转债持仓上限

| 对标的把握度 | 建议上限 |
|-------------|---------|
| 不确定（初学者） | **10%** |
| 较有信心 | 30% |
| 极有信心 | 50% |

> **最低持有多只**：上限30%意味着至少持有3~4只，通常可容10只左右，有效分散风险。

#### (3) 整体仓位控制（"高筑墙、广积粮、缓称王"）

| 整体仓位区间 | 操作策略 |
|-------------|---------|
| **0~30%** | 满足建仓线即可买入，快速建底仓 |
| **30~60%** | 放慢节奏，或只到加仓线才买 |
| **60~100%** | 严格管控，不到重仓线不买，或大幅下调三条线 |

> 也可以按 **30%:30%:40%** 的比例分配整体建仓节奏。

### 5.4 如何根据个股资质调整三条线？

| 调高（放宽）条件 | 调低（收紧）条件 |
|----------------|----------------|
| 股性强、波动大 | 股性弱、波动小 |
| 题材突出、业绩好 | 无题材、业绩差 |
| **PB高**（下调空间大） | **PB<1**（下调无空间，致命伤） |
| 有回售条款 | 无回售条款 |
| 盘子小 | 盘子大 |
| 到期价值高 | 到期价值低 |
| 有转股愿望 | 转股无动力 |

**实战示例**（2014年3月安道全团队设置）：

| 转债 | 现价 | 建仓线 | 加仓线 | 重仓线 | 备注 |
|------|------|--------|--------|--------|------|
| 博汇 | 104.94 | 107.00 | 103.00 | 100.00 | 到期/回售/面值 |
| 歌华 | 97.79 | 100.00 | 96.71 | 92.16 | 面值/年化4%/年化6% |
| 石化 | 103.99 | 100.00 | 97.79 | 92.83 | 面值/年化4%/年化6% |
| 中海 | 91.00 | 100.00 | 95.02 | 89.62 | 面值/年化4%/年化6% |

### 5.5 三条安全线的哲学意义

- **高筑墙** → 安全（重仓线几乎封死所有亏损可能）
- **广积粮** → 资金（保留预备队，捕捉更低价格）
- **缓称王** → 仓位控制（从容不迫，好整以暇）

## 六、面值-高价折扣法（卖出策略补充）

在"三条安全线"买入基础上，卖出端使用**面值-高价折扣法**：

- **核心规则**：当可转债价格创出新高后，从最高点回落一定幅度（如10元/10%），即卖出
- **改进意义**：纯面值-130法只能吃到30~40%利润，高价折扣法可多次兑现，化波动为收益
- **风险增加**：不过1%，收益可能翻番
- **实战效果**：避免唐钢转债（开盘130-150买入，最后111元退市）等悲剧

## 七、对"双低策略"持有人的重要补充提示

> 基于安道全书中的核心论点，以下几条值得已有双低策略仓位（15,000元）的稳盈君特别关注：

1. **双低策略的本质是安道全"面值法"的量化延伸**：双低 = 低价格（靠近面值）+ 低溢价率（转股容易），本质上是在寻找"安全+有弹性"的标的。

2. **仓位管理是双低策略的命门**：安道全三条安全线告诉我们——绝不是简单地"筛出来就满仓"。用3:4:3比例分批买入，单只上限10~30%，比一次性建仓安全得多。

3. **极端市场是朋友不是敌人**：2008年南山转债82.7元 → 199元的案例说明，市场恐慌时才是重仓良机。双低策略持有者应保留预备队"广积粮"。

4. **强制赎回是终点而不是全部**：大部分双低转债最终以130元以上退市/强赎结束，但过程中可能有巨大波动（如2007年招商地产转债0.73年完成从100→350）。"不动如山"的定力比筛选条件更重要。

5. **PB不可忽视**：安道全明确指出PB<1的可转债下调空间为零——这也是双低筛选中容易被忽略但至关重要的指标。

> "可转债，是一个可以让投资者安心睡觉的投资工具。"
