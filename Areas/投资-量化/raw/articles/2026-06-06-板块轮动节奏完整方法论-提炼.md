# 板块轮动节奏完整方法论 · 精华提炼

## 核心观点

1. **板块轮动的两大源头**：市场心理周期（情绪兴奋→消退的规律性重复）+ 行业周期与政策导向（经济周期、监管政策、产业兴衰）
2. **板块强度→看三个指标**：涨停数量、内部结构紧密程度（龙头涨停后龙2几分钟内跟上）、龙头稳定性（出现抢龙头=接近尾声）
3. **退潮终极信号**：板块最冷门股票开始上涨 = 板块见顶信号，该跑

## 可操作原则

### 板块轮动类型
- **主线**：建制完整（龙头+中军+先锋+补涨），持续性最强
- **次主线**：结构略弱，多为双主线交替
- **支线**：板块效应不强，多为助攻
- **一日游**：消息刺激+一字板，观察几天才有龙头

### 板块强度判断
- 竞价涨幅前列 + 开盘半小时3只以上底部放量上攻 = 强
- 早盘一字板数量多 = 强
- 百万手大单封死 = 给板块打空间
- 板块内大盘蓝筹股异动往往是轮动先兆
- 板块内有不断趋势新高的股票 = 大资金已潜伏

### 7种不能做的场景
1. ⛔ 板块高潮 → 次日必分歧，放弃接力
2. ⛔ 分歧后回流 → 别人拿先手你接盘
3. ⛔ 高位龙头 → 没有新故事不接力
4. ⛔ 连续缩量加速 → 危险信号
5. ⛔ 龙头还强就提前切补涨 → 节奏错误
6. ⛔ 高潮次日接力 → 接盘无疑
7. ⛔ 连续跌停时博反弹 → 切忌

### 轮动信号识别
- **见顶→恐慌→惯性下跌→情绪修复**（第4-5天可关注）
- **二轮上涨**：力度减弱、速度加快，后期谨慎
- **产业上下游传导**：下游产品→中游加工→上游原材料
- **政策传导**：国家政策→相关行业板块
- **比价效应**：龙头涨→同属性补涨
- **拆台（跷跷板）**：大蓝筹与中小创互相制约→谨慎/空仓
- **协同作战**：多板块同涨→可积极参与

### 大盘环境
- 成交量 < 8000亿 → 难持续
- 成交量 > 1万亿 → 易出主线
- 指数在5日线上方 = 容易赚钱；下方 = 容易亏钱
