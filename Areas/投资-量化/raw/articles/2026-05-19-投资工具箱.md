## 五、自定义指标库

### 💰 真实盈利

```
真实盈利 = 每股企业自由现金流 / 每股收益
```

| 判断标准 | 含义 |
|----------|------|
| > 1 | 经营现金流覆盖净利润，佳 |
| < 0.5 | 利润是纸面富贵，警惕 |

> 可手动检查：经营现金流/净利润 > 1 为佳

---

### 📐 中性ROE

```
中性ROE = TTM(归属于母公司所有者的净利润) / AvgQ(归属母公司股东权益合计, 4, 1)
```

| 判断标准 | 含义 |
|----------|------|
| > 15% | 赚钱效率高 |
| < 5% | 盈利能力弱 |

---

### 📊 股价偏离度

```
股价偏离度 = 后复权收盘价 / 20日复权均价 - 1
```

| 判断标准 | 含义 |
|----------|------|
| < 8% | 股价平稳，未被高估 |
| > 15% | 短期涨幅过大，谨慎追高 |

## 七、每周复盘 SOP

### 复盘内容

1. **持仓健康度检查** — 运行 `stock_analysis.py`，浮亏扩大？浮盈守住？
2. **双低池子更新** — 运行 `cb_dual_low_scan.py`，有无新机会？持仓债调仓？
3. **市场环境判断** — 大盘情绪、可转债市场整体中枢、热点方向
4. **策略有效性评估** — 本周收益 vs 沪深300，双低 vs 大盘
5. **下周操作计划** — 明确 1~3 个具体操作，不开新仓

- 双低进攻仓：XXXX

🎯 下周计划
1. [操作] 具体标的 + 条件
2. [操作] 具体标的 + 条件
3. [操作] 具体标的 + 条件（可选）

## 八、微信文章分析 SOP

### Step 1：抓取
```bash
python3 /root/.hermes/scripts/wechat_fetch.py --url <链接>
```
保存至：`articles/YYYYMMDD_标题.json`

### Step 2：可行性评估（6条）
1. 逻辑是否成立 — 因子有没有经济学/行为金融学依据
2. 回测数据是否可信 — 年化>30%需警惕幸存者偏差/过拟合
3. 数据能否获取 — 散户能拿到所需数据吗
4. 资金量能否执行 — 9.4万够不够分散
5. 执行成本 — 手续费、滑点、时间精力
6. 是否符合用户约束 — 不止损、调仓换仓为主

### Step 3：评分与结论
- 评分维度：逻辑完整性 / 数据可获取性 / 落地可行性 / 互补性
- 结论：**✅采纳** / **⚠️借鉴** / **❌不适用**

### Step 4：归档
- 采纳 → 存入 `strategies/`
- 借鉴 → 提取因子，融入现有策略
- 不适用 → 记录原因
