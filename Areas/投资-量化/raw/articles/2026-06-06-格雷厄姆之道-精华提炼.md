# 格雷厄姆之道 精华提炼

| 原则 | 原文 | A股化翻译 |
|:--|:--|:--|
| 安全边际 | "用4毛钱买值1块钱的东西" | PE<10倍且PB<1倍才有安全边际 |

## 二、烟蒂股策略（中国化改造）

格雷厄姆的原始方法在美国已经很难用了（信息太透明），但在A股依然有效。

### 筛选条件（A股版）
```
1. PE < 10倍
2. PB < 1倍（破净）
3. 股息率 > 3%
4. 市值 > 50亿（排除垃圾壳股）
5. 经营现金流 > 净利润（赚的是真钱）
6. 负债率 < 60%
```

### 操作规则
- 分散买入至少10只（格雷厄姆的核心要求）
- 任何一只涨超50%就卖
- 持有时间最长不超过2年
- 2年没到目标也卖——说明判断错了

### 为什么A股有效
A股散户多、机构短期化、信息不透明——这些恰恰是格雷厄姆式价值投资的最佳土壤。当一只股票连续破净、PE在个位数的时候，说明市场已经**过度悲观**——这正是邓普顿说的极度悲观点。

## 三、格雷厄姆的防御型投资者策略

适合没时间盯盘的散户：

```
1. 资金分两部分：
   - 50% 宽基指数基金（沪深300/中证500）
   - 50% 高评级债券（国债/AAA企业债）
2. 每年再平衡一次（哪个涨多了卖一点补另一个）
3. 每季度查一次持仓，平时不看
4. 年化预期：8~12%
```
