# 核心笔记：量价/分时/盘口技术体系 精华提炼

## 一、核心原则（必记）

### 1. 量价定性三原则
| 原则 | 内容 | 应用 |
|:--|:--|:--|
| 高量=拐点 | 高量柱是主力异动，代表方向可能改变 | 当天不操作，观察3天定性 |
| 缩量定方向 | 缩量阴线=洗盘（跌不动）；缩量阳线=拉升（还有空间） | 缩量阴线不抄底，等次日；缩量阳线持股 |
| 放量见分晓 | 放量意味着分歧，分歧处要么建仓要么出货 | 高位放量=出货，低位放量=见底 |

### 2. 高量柱守则（全书最高频概念）
```
1. 高量第1天 → 观察
2. 高量第2/3天支撑线上方 → 阳线上，阴线观察
3. 高量破，有灾祸 → 清仓
4. 高量不分阴阳，成交量不看颜色
5. 高量定三天，三天定性
6. 以当天高量实体低点为支撑位
7. 以过往高量实体高低点为压力位
8. 下跌趋势第一次高量 → 诱多，观察3天
```

### 3. 量价16种关系（最实用的部分）
| 状态 | 含义 |
|:--|:--|
| **放量大涨**（量价齐升） | 趋势延续，主力建仓初期 |
| **缩量大涨** | 主力控盘高，一致看涨，还要涨 |
| **放量大跌** | 高位出货信号，卖压极大 |
| **缩量大跌** | 恐慌效应，还会跌 |
| **放量滞涨**（价不涨量放大） | 抛压重，见顶信号 |
| **缩量微跌**（价小跌量缩小） | 洗盘，洗完继续拉 |
| **放量微跌**（价小跌量放大） | 有承接，见底信号 |
| **缩量上涨**（量缩价涨） | 空方减弱，等加速 |
| **缩量下跌**（量缩价跌） | 无人接盘，继续跌 |

## 二、分时图八类形态（实用框架）

### 识别主力意图的8种分时形态

| 类型 | 具体形态 | 识别特征 | 操作 |
|:--|:--|:--|:--|
| **建仓** | 脉冲建仓/冲击建仓/攻击建仓 | 尖角波+放量+大单间歇性买入 | 底部出现可跟随 |
| **试盘** | 向上试盘/向下试盘 | 瞬间拉升/打压后迅速归平静 | 测试阻力/支撑，关注后续 |
| **诱多** | 早盘诱多/盘中诱多/尾盘诱多 | 开盘高开低走/盘中突然拉升/尾盘偷袭 | 高位出现=出货前兆，减仓 |
| **洗盘** | 强势洗盘/弱势洗盘 | 高开低走缩量/平缓回落缩量 | 缩量洗盘是机会 |
| **承接** | Y字形结构 | 价位线V字+量能低谷=红酒杯形 | 底部反复承接=跌不动 |
| **护盘** | 尾盘承接 | 尾盘加速下跌后Y字形拉起 | 洗盘尾声，即将反转 |
| **诱空** | 盘中诱空/开盘诱空 | 早盘拉升或下跌→震荡造恐慌→尾盘拉起 | 洗盘末期=理想买点 |
| **拉升** | 涨停板拉升 | 高开高走/突然拉升+流畅封板 | 最值得参与的位置 |

### ⭐ 核心买点信号：诱空形态（重点关注）
- **盘中诱空**：开盘一波快速拉升或下跌 → 盘中震荡造恐慌 → 收盘站上均价线
- **位置**：洗盘末期，主力即将开始新一轮拉升
- **操作**：等待诱空形态出现后，次日确认再入场

## 三、保护线系统（风险控制）

### 止损保护线
- 设置在最近支撑位置（筹码峰峰谷）
- 拉升中途介入：止损线=最近阳线最低点 + 最近筹码峰峰谷，**两层击穿必须止损**

### 止盈保护线（移动止盈）
- 设置在最近上涨阳线 / 筹码峰峰谷（取高者）
- 机构参与度高的股票：用筹码峰波谷做止盈保护，可抓整个趋势

## 四、趋势股 vs 短线股（重要概念区分）

| 维度 | 趋势股（机构股） | 短线股（主力股） |
|:--|:--|:--|
| 股东结构 | 公募/北向/社保/险资 | 游资/个人 |
| 行情类型 | 趋势性行情（数月） | 一波流+龙回头 |
| 防守方法 | 有效跌破60日线 | 跟主力做完一波清仓 |
| 止盈方法 | 趋势背离形成卖点 | 破5日线走人 |

> **对我当前持仓的意义：** 长电/移动（趋势股，看60日线）；灿芯/拉普（短线/题材股，破5日/10日线就走）

## 五、均线共振手法（波段操作核心）

**原理：** 5/10日线金叉后，股价已涨了一段，此时**不能追高**，等回踩5/10日线确认后再入场。

**核心买点：** 金叉后缩量回踩不破均线 → 放量阳包阴 → 买入

**级别对应：** 
- 5/10日线共振：1~3天
- 60/120日线共振：20天以上

### 见顶信号（出现即减仓）
| 形态 | 特征 |
|:--|:--|
| 黄昏十字星 | 阳线→十字星→阴线，第三根深入第一根实体 |
| 乌云压顶 | 阳线后高开低走阴线，收盘深入阳线实体 |

| 高位平顶/圆顶/塔顶 | 多根K线顶部在同一水平 |

## 七、缩量法则（号称100%验证）

### 缩量阴线风控法
- **原理：** 缩量只对应洗盘或拉升。缩量阴线=洗盘没完成
- **验证：** 第二天盘中至少跌破当天收盘价（沪指13次100%验证）
- **操作：** 当天不抄底，等第二天盘中再说
- **声明：** 下跌行情有效性高，上涨行情慎用

### 缩量阳线持股法
- **原理：** 缩量阳线=拉升没完成
- **验证：** 第二天盘中至少突破当天收盘价（沪指11次100%验证）
- **操作：** 当天不卖出，等第二天拉升再卖
- **声明：** 上涨行情有效性高，下跌行情慎用

## ⚠️ 忽悠内容识别

### ❌ 绝对垃圾（不要信）
1. **盘口数字密码**（111=要要要、444=死死死、555=捂捂捂等）
   - 纯属玄学。20年前股吧论坛的迷信产物，**没有任何统计支撑**
   - 如果有人靠这个赚钱，彩票分析师早就是全球首富了

2. **"胜率95%/96%"** 的宣称（阴线反包法95%、龙回头96%）
   - 这些数字没有任何回测基础，纯营销话术
   - 任何一个策略如果真有95%胜率，年化至少500%，作者早去华尔街了

3. **"主力意图"过度解读**
   - 书中几乎所有分析都建立在"看穿主力在想什么"的假设上
   - 现实是：多数盘口异常由算法交易、机构执行、做市商行为造成，不是"主力故意让你看见"

### ⚠️ 部分有用但被包装过度
