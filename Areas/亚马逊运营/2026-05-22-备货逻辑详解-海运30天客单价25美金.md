## 一、核心挑战：物流周期与打造节奏的Gap

### 时间线分析

```
T-37天  → 发货（装柜/入仓）
T-30天  → 到达亚马逊仓库
T-23天  → 上架可售
T-0天   → 开始螺旋打造

物流周期：海运30天 + 入仓上架7天 = 37天
```

### 问题所在

| 阶段 | 时长 | 挑战 |
|------|------|------|
| 发货等待 | 30天 | 这段时间能做啥？ |
| 上架后 | ?天 | 不知道销量，备多少货？ |
| 打造过程 | 60-90天 | 需要持续有货 |
| 补货周期 | 37天 | 一旦断货=等37天才有货补上 |

## 二、备货量计算公式

### 老魏原话（第20讲案例）

> "因为他选择的是拿货成本只有10元左右的低价产品，而新店铺的可用仓容又被亚马逊系统限定为1000，考虑到成本不高、风险不大，他就直接一次性发了1000个货到FBA仓库。"

### 备货量三要素

```
备货量 = max(可用仓容, 安全库存天数 × 预期日销量)
```

### 针对$25客单价产品的计算

#### 场景假设
```
客单价：$25
预期日销量（螺旋期）：30-50单
亚马逊新品仓容限制：500-1000
头程物流：30天
上架：7天
打造周期：60-90天
```

#### 第一批货计算

| 计算维度 | 数值 | 说明 |
|----------|------|------|
| 安全库存 | 30天 | 防止断货的缓冲 |
| 预期日销量 | 40单 | 保守估计 |
| 最低备货量 | 500单 | 37天物流周期 |
| 建议备货量 | **600-800单** | 考虑销量增长 |

**关键**：不要只看当前销量，要预估30天后（货到时）的销量

#### 第一批货分批发货策略

```
第一批 A批：300单 → 海运30天
第一批 B批：300单 → A批上架后补发（空运备急）
```

## 三、断货前应对策略（老魏第17讲核心）

### 断货对Listing的伤害

> "遭遇断货后，Listing权重会大幅下降，当下一批货入仓上架后，往往卖家会发现销量比断货前差很多"

### 三步应对法

#### 第一步：小步快跑涨价（减少断货时间）

**原理**：
- 涨价 → 抑制销量 → 减少断货时间
- 涨价 → 赚取超额利润 → 弥补断货损失

**操作**：
```
平时节奏：每3-7天涨$0.5-$1
断货前节奏：每天涨$0.2-$0.5（多次）

举例：原价$12.99
第1天：$13.49
第2天：$13.99（销量降了就停）
第3天：$14.49（如果还没断）
```

**注意**：单次涨幅不能太大，否则会被亚马逊判定操纵排名

#### 第二步：高价自发货跟卖（防止僵尸Listing）

**原理**：
- 断货后Listing无人看管 → 被恶意卖家篡改
- 自发货跟卖 → 保持Listing在售状态

**操作**：
```
FBA售价：$25.00
跟卖价格：$30.00-$33.00（加$5-$8）

目的：
1. 防止Listing被抢走
2. 多渠道配送时额外收费覆盖成本
```

#### 第三步：降低广告预算（省钱）

```
断货前 → 逐步降低广告竞价和预算
断货期间 → 暂停或极低预算
补货上架后 → 恢复广告投放
```

## 四、补货上架后运营策略（老魏第17讲）

### 核心逻辑

> "用断货前赚取的超额利润，补贴补货后的运营打造"

### 两步走

#### 第一步：适当降价

```
断货前售价：$28.99（涨价后）
补货上架价：$24.99（比正常价低$3-4）

目的：
- 价格敏感性激活销量
- 转化率提升 → 权重恢复
```

#### 第二步：提高广告竞价和预算

```
补货上架后：
- 竞价：提高到断货前水平
- 预算：比断货前略高10-20%

配合降价 → 广告导入流量 + 低价带动转化
```

## 五、针对$25客单的具体备货方案

### 方案一：保守型（新手推荐）

```
第一批：500单（海运）
  - 预计可用天数：500 ÷ 30单/天 ≈ 17天
  - 触发补货点：库存剩200单时
  
第二批：800单
  - 400单海运 + 400单空运
  - 海运保底，空运应急
  
空运备货：200单（随时待发）
  - 空运周期：7-10天
  - 断货前的最后防线
```

### 方案二：激进型（有资金实力）

```
第一批：1000单（海运）
  - 发货后30天才上架
  - 上架后预计销量爬坡：
    - 第1周：20单/天
    - 第2周：30单/天
    - 第3周：40单/天
    - 第4周：50单/天
  - 到货第3周开始补货
  
第二批：1500单
  - 800单海运 + 700单空运
```

### 方案三：老魏学员案例模板

> 参考第20讲学员做法：
> 1. 先发满仓容限制的量（不激进，留有余地）
> 2. 用海运慢船降低物流成本
> 3. 仓容扩大后，按仓容上限补货
> 4. 保持充足库存 = 不用担心断货 = 可以慢慢涨价

## 六、物流方式选择

### 三种物流对比

| 物流方式 | 时效 | 成本 | 适用场景 |
|----------|------|------|---------|
| 海运慢船 | 30-35天 | 最低 | 第一批货、批量补货 |
| 海运快船 | 18-22天 | 中等 | 常规补货 |
| 空运 | 7-10天 | 高3-5倍 | 应急补货、断货前追加 |
| 快递 | 5-7天 | 最高 | 紧急补货 |

### 推荐配比（针对$25客单价）

```
日常补货：70%海运 + 30%空运备用
安全库存：保证有200单空运货随时可发
```

## 七、仓容管理

### 新品期仓容限制

| 店铺类型 | 初始仓容 | 触发条件 | 扩大后 |
|----------|---------|---------|-------|
| 新店 | 500-1000 | - | 动销率高 |
| 老店 | 2000-5000 | IPI > 400 | 销量稳定 |
| 大卖 | 10000+ | 绩效优秀 | 持续扩展 |

### 仓容扩大策略

老魏第20讲案例：
> "销售到第二个月，因为动销率不错，他的FBA仓容被打开，从1000变成了3000"

**关键**：保持高动销率 = 库存一直在卖 = 仓容不断扩大

## 九、核心原则总结

| 原则 | 说明 |
|------|------|
| **宁可多备，不可断货** | 断货一次，权重损失2-3周才能恢复 |
| **按仓容发货** | 新店先发满限制量 |
| **海运为主，空运备用** | 降低成本，保证应急 |
| **断货前小步快跑涨价** | 减少断货时间，赚取超额利润 |
| **补货后降价恢复权重** | 用断货前的利润补贴 |
| **保持空运库存** | 永远有200单可快速补上 |

## 十、关键公式

```
备货量 = (物流周期 + 安全库存天数) × 预期日销量

断货预警点 = 日销量 × 补货周期 + 最低库存

补货周期 = 海运30天 + 上架7天 + 缓冲3天 = 40天

$25产品断货预警 = 日销量 × 45天
```

## 十一、常见问题

### Q：$25产品海运成本高吗？
A：看重量体积，通常$25产品重量500g内，海运成本约¥15-20，占售价比例可接受

### Q：第一批没卖完可以发第二批吗？
A：可以，新品期仓容会随动销率扩大，不用担心爆仓

### Q：空运成本太高怎么办？
A：空运只备200单应急即可，大部分靠海运。断货损失 > 空运额外成本

### Q：断货后可以直接降价冲量吗？
A：可以，但要配合广告加大投放。单纯降价效果有限
