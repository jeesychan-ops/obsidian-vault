# 选品决策SOP：市场筛选 + 七步漏斗法

## 核心框架：两层决策

```
第一层：选赛道（4选1）→ 第二层：7步漏斗 → 五维评分终审 → 飞书记录
```

---

## 🚪 第一层：选赛道（卖家精灵4种模式 → Sorftime实现）

**先问自己：我是谁？我能做什么？**

| 卖家类型 | 筛选模式 | 核心条件 | Sorftime工具 |
|---------|---------|---------|-------------|
| 有资金，想冲规模 | 爆款市场 | 月销>3000, 上架>1年 | category_basic_info + top100 |
| 贸易型中小卖家 | 潜力市场 ⭐ | 月销<600, 上架≤6个月, 增长>10% | top100 + review_distribution |
| 工厂型（有改造能力） | 未被满足的市场 ⭐ | 月销>300, 评分<3.7 | top100 + brand_market_share |
| 低资金想试错 | 不压库存的市场 ⭐ | 月销>300, FBM为主 | top100 |

> 工具：跑 category_basic_info 看增长率，跑 review_distribution 看上架时间，跑 brand_market_share 看评分

## 📋 第二层：7步漏斗法（全部工具替换为Sorftime）

### Step 1：画理想产品画像
| 指标 | 标准 | 工具 |
|------|------|------|
| 重量/体积 | <2kg, 不超标FBA | supply_chain_1688 |
| 客单价 | $15–75 | price_distribution |
| 季节性 | 四季稳定或旺季可预判 | category_basic_info |
| 法律风险 | 无专利/商标 | 人工核查 |
| 品牌依赖度 | 弱（工具/家居>数码） | top100品牌集中度 |
| 市场容量 | 月销足够支撑备货 | category_basic_info |

### Step 2：找产品灵感
- Amazon Best Sellers
- Sorftime category_top100 看哪些细分上榜容易
- 差评挖需求（竞品1-3星评论找痛点）
- 1688新品（supply_chain_1688看起量新品）

### Step 3：关键词验证
- Sorftime keyword工具 查搜索量和竞争度
- 判断：月搜索量>1000 + 相关词10+ = 有流量基础

### Step 4：市场分析
| 维度 | Sorftime工具 | 判断标准 |
|------|------------|---------|
| 市场容量 | category_basic_info | Top100月总销>5万单 |
| 垄断程度 | brand_market_share | HHI<2000 |
| 新品占比 | review_distribution | 上架<6个月>20% |
| 评论门槛 | review_distribution | 头部评论<200=新手友好 |
| 价格分布 | price_distribution | 有价格空档 |

### Step 5：竞品 VOC 分析
- 差评TOP3：评分最低的几个ASIN → 客户抱怨=你的机会
- 卖点布局：哪些还没被强调
- 视觉差距：哪些风格没人做
- 变体策略：哪些组合还空着

### Step 6：利润测算
| 项目 | 工具 |
|------|------|
| 1688拿货价 | supply_chain_1688 |
| FBA费用 | Sorftime FBA计算器 |
| 毛利率 | ≥30% 才入场 |

### Step 7：小单验证
```
首批：200–500件
广告预算：尾部ASIN 25% | 长尾词 50% | 自动广告 25%
周期：7天 / 14天 / 30天
指标：曝光↑ CTR>0.5% CVR>10% ACOS<35% 退货率<5%
通过：14天稳定出单 → 五维终审
不通过：30天无起色 → 重新选品
```

## ✅ 五维评分终审

| 维度 | 分值 | 通过门槛 |
|------|------|---------|
| 市场规模 | 20分 | 满分=月销大 |
| 增长潜力 | 25分 | 满分=高增长+稳定 |
| 竞争烈度 | 20分 | HHI<1500=满分 |
| 进入壁垒 | 20分 | 评论<100=满分 |
| 利润空间 | 15分 | 毛利率>30%=满分 |

**≥60分入场，<60分换方向或强差异化**
