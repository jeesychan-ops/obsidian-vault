# 亚马逊运营高频问题 FAQ

## 选品类

### Q1: 选品三维度具体用什么数据/工具评估？

| 维度 | 工具/数据源 | 标准 |
|------|-----------|------|
| 利润率 | 成本计算器（Sorftime / 卖家精灵） | 毛利率 > 30%，净利 > 15% |
| 竞争度 | Sorftime 反查 BSR 前 10 | 新卖家 2-3 个 = 有机会 |
| 需求稳定性 | Google Trends + ABA 月搜量 | 三年曲线平稳，非脉冲 |

详见：[[选品三维度法]]

### Q2: 什么样的产品适合万词打法？

**非标品、功能分散型产品万词打法效果最佳**：
- ✅ 服装/家居/3C 配件/宠物用品（搜索极度分散）
- ✅ 功能型产品（有大量细分场景词）
- ❌ 极标准化产品（大词垄断，无长尾空间）

### Q4: ACOS 太高怎么降？

**ACOS 优化三步法**：
1. **关键词升级**：把低竞争跑出的好词往中高竞争搬（规模效应）
2. **长尾词替换**：大词 ACOS 高 → 换场景型长尾词（CPC 低 50-80%）
3. **落地页优化**：Listing CVR 提升 → 分母变大 → ACOS 降低

> 案例：空气炸锅衬纸，"air fryer liners" ACOS 28% → 布局 "air fryer liners for ninja foodi dual"（场景型），ACOS 14%，销量增长 65%。

详见：[[关键词竞争度决策框架]] + [[万词打法]]

### Q5: 关键词竞争度怎么判断要不要升级？

**三层判断工具（升级前必看）：**
1. ABA 数据：点击份额被头部锁死 = 没空间
2. 前台竞争：品牌集中度高 = 价格战
3. 广告数据：CPC 涨 + 转化跌 = 竞争加压中

**升级失败三信号（出现任意一条停止升级）：**
- CVR 明显低于类目基准
- 预算撑不起分层结构
- CPC 持续涨 + 转化持续跌

详见：[[关键词竞争度决策框架]]

### Q6: 站外引流什么时候开始做？

**建议时机**：有稳定自然流量后（评论 ≥ 20，BSR 稳定），否则站外引流效率低。

**最优打法**：
1. TikTok/Instagram 内容种草 → 挂亚马逊链接
2. Amazon Attribution 追踪站外流量
3. 站外成交进店 → SD 再营销（ACOS 仅 8%）
4. 品牌词搜索量一周涨 340%，自然排名冲到前 5

## 流量类

### Q7: 自然流量怎么快速增长？

**核心路径：广告带自然、自然带爆单**

```
广告长尾词高转化
  → A9 权重提升
  → 长尾词自然排名上升
  → 大词自然排名跟着涨
  → ACOS 逐步降低
  → 爆单飞轮
```

**实操：关键词上首页 3 个打法**
1. 广告点击 → 转化 → 算法认定高质量 → 给自然排名
2. QA 植入核心场景词 → COSMO 抓取 → 场景流量
3. 评论关键词植入 → Alexa 理解产品属性 → 品类推荐

### Q8: Alexa/COSMO 时代 Listing 怎么写？

**旧逻辑**：关键词密度
**新逻辑**：**语义完整度**——是否完整回答四个问题：
1. 这是什么？
2. 适合谁？
3. 解决什么痛点？
4. 为什么选你？

**落地**：
- 五点描述：每条对应一个场景/人群/痛点
- A+ 内容：使用场景图 + 人群对比图
- QA：植入高频场景问题（Alexa 会基于这些回答用户）
- 评论：引导用户写带场景关键词的真实评论
- 外部内容：Reddit/Medium 布局 AI 外部数据源
