## 运营阶段诊断

### 初创期

特征：
- Listing 新上架
- 评论数量少
- 关键词排名靠后

核心任务：
- Listing 优化
- 评论积累
- 基础关键词收录

### 成长期

特征：
- 稳定出单
- 部分关键词有排名
- 利润压力显现

核心任务：
- 关键词层级提升
- ACOS 优化
- 自然流量占比提升

### 成熟期

特征：
- 销量稳定
- 品牌有一定认知
- 面临同质化竞争

核心任务：
- 品牌建设
- 产品线扩展
- 多渠道布局

### 突破期

特征：
- 增长遇到瓶颈
- 竞争加剧
- 成本上升

核心任务：
- 差异化升级
- 供应链优化
- 精细化运营

## AI 在亚马逊运营中的应用

### 全链路覆盖

```
选品 ──► Listing ──► 广告 ──► 客服 ──► 数据分析
  │        │          │         │          │
  ▼        ▼          ▼         ▼          ▼
 AI辅助   AI生成    AI优化   AI回复   AI分析
```

### AI 应用场景

| 环节 | AI 应用 |
|------|---------|
| 选品 | 市场分析、竞品调研、趋势预测 |
| Listing | 标题/五点生成、关键词优化、A+ 内容 |
| 广告 | 关键词策略、出价优化、预算分配 |
| 客服 | 自动回复、差评处理、FAQ 生成 |
| 数据 | 销售预测、利润分析、竞品监控 |

## 运营全局观

### 运营漏斗

```
         曝光
           │
           ▼
         点击 ──────► 流量获取
           │              │
           ▼              │
         转化 ──────► 转化优化
           │              │
           ▼              │
         复购 ──────► 用户运营
           │
           ▼
         利润
```

### 核心指标体系

| 指标 | 定义 | 优化方向 |
|------|------|----------|
| 曝光量 | Listing 被展示次数 | 关键词收录、权重提升 |
| 点击率 | 点击/曝光 | 主图、价格、评分 |
| 转化率 | 订单/点击 | Listing 质量、评论 |
| 复购率 | 重复购买占比 | 产品质量、客服 |
| 利润率 | 利润/销售额 | 成本控制、定价策略 |
