## 二、流量：四维算法协同

亚马逊目前有四个算法同时运行，它们决定了流量的分配：

1. **A9/A10**：传统搜索排名，决定「搜这个词，谁排前面」
2. **COSMO**：意图理解，决定「搜的人是不是对的人」
3. **Rufus**：AI 主动推荐，决定「没搜的人怎么被种草」

**飞轮效应**：
```
站外引流 → 站外成交 → 站内权重提升
  ↑                              ↓
广告转化 ← 精准长尾词 ← 自然排名提升（广告带自然）
  ↑
高转化 → 算法给更多自然流量（自然带爆单）
```

详见：[[四维算法协同体系]]

## 三、广告：三层架构 × 四维算法协同

### 广告三层架构
1. **第一层（自动广告）**：跑词 + 养算法，低竞价
2. **第二层（手动精准）**：核心转化词，精准匹配
3. **第三层（品牌/DSP）**：拉新 + 品牌保护，成熟期开启

### 意图分层投放（基于四维算法）
- 信息型词（how to / best）：低竞价跑数据
- 对比型词（vs / compare）：ASIN 定投竞品
- 场景型词（for / with / when）：COSMO 长尾词重点布局
- 购买型词（buy / deal）：高竞价守位

### 全渠道组合
**SP+SD+SB+DSP 组合才能实现指数级增长**，单一 SP 有明显天花板。

详见：[[广告策略]]

## 四、关键词：万词打法 + 竞争度框架

### 万词打法：流量金字塔
- 塔尖 1%：品牌词 + 高转化精准词（防守）
- 腰部 20%：场景 + 属性 + 人群词（放量核心）
- 基部 79%：细分长尾词（拓词 + 养权重）

### 竞争度四层迁移路径
**低竞争找机会 → 中竞争做承接 → 中高竞争做放大 → 高竞争做卡位**

节奏：7 天验证趋势 → 14 天做升级/降级动作

## 五、站内流量：转化率优先

亚马逊流量分配的底层逻辑：**转化率优先，滚雪球增长**

| 阶段 | 流量来源 | 策略 |
|------|---------|------|
| 新品期 | 广告为主 | 冷启动，积累数据 |
| 成长期 | 广告 + 自然推荐 | 提升转化，降低 ACOS |
| 成熟期 | 自然为主 | 降低广告依赖 |

详见：[[站内流量]]

## 六、竞品分析：六维度法

1. 销量估算（BSR → 日均单量）
2. 评论分析（评分分布 + 评论增长率）
3. 流量来源反查（ABA → 流量词分布）
4. 广告打法推断（搜索词报告 → 匹配方式）
5. 定价策略（价格带 + 促销节奏）
6. 差异化机会（功能/场景/人群空缺）

详见：[[竞品分析]]

## 七、运营节奏总结

```
选品 → Listing 优化 → 冷启动广告
  → 成长期：万词打法 + 竞争度迁移
  → 成熟期：降广告依赖 + 站外引流反哺
  → 持续优化：四维算法协同
```

**核心公式**：
> 爆款 = 选品（40%）+ Listing 质量（30%）+ 广告效率（20%）+ 评价积累（10%）
