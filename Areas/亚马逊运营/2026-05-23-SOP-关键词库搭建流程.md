# 亚马逊关键词库搭建 SOP — 四步循环建设法

## 核心理念
> 关键词库不是一次性建完的，而是从竞品出发 → 持续积累 → 循环验证 → 动态淘汰的闭环系统。

## 流程概览
```
竞品池确定 → Sorftime批量反查 → 分层分类 → 入库+绑定广告
```

---

## Step 1: 竞品池确定（来源选择）

**竞品池构成（10-15个ASIN）：**

| 类型 | 数量 | 选择标准 |
|------|------|---------|
| 核心竞品 | 5-8个 | 价格相近、销量稳定、排名靠前 |
| 新品潜力股 | 3-5个 | New Release排名好、评分4.0+、上架<6个月 |
| 品牌竞品 | 2-3个 | 品牌知名度高，观察品牌词流量来源 |

**ASIN来源：**
- 搜索核心词（#1/#2/#3推荐位竞品）
- BSR榜单Top 50
- 广告位竞品（广告出现频率高的ASIN）

---

## Step 2: 批量关键词反查（Sorftime MCP）

**工具：** Sorftime MCP → `sorftime_reverse_keywords` 或手动批量查询

**操作流程：**

1. 将竞品ASIN列表整理成文本（每行一个）
2. 对每个ASIN执行关键词反查，获取：
   - 搜索词列表
   - 月搜索量
   - 搜索量指数（相对值）
   - 排名趋势（升/降/平稳）
3. 合并所有竞品词表，去重

**反查维度（每个ASIN必查）：**
- 自然搜索词（自然流量来源）
- 广告搜索词（SP广告投放词）
- 关联流量词（与本品关联购买的ASIN词）

**输出：** 原始词表（Excel/CSV，字段：关键词 | 搜索量 | 竞争度 | 来源ASIN）

---

## Step 3: 关键词分层与分类

### 3.1 按搜索量分层

| 层级 | 月搜索量 | 特征 | 用途 |
|------|---------|------|------|
| 核心大词 | >10,000 | 流量大、竞争激烈、CPC贵 | 广告+Listing标题辅助 |
| 二级词 | 1,000-10,000 | 转化好、竞争中等 | 手动广告精准主投 |
| 长尾词 | 100-1,000 | 竞争小、转化精准 | 手动精准+ASIN定投 |
| 碎片词 | <100 | 极低搜索量但真实意图 | 否定词参考/自动广告跑出 |

### 3.2 按匹配意图分类

| 类型 | 特征词 | 说明 |
|------|--------|------|
| 品牌竞品词 | [竞品品牌名] + 产品词 | 拿来抢对手流量的 |
| 品类词 | 大词/类目名 | 自然流量主入口 |
| 属性词 | 材质/颜色/尺寸/功能 | 精准需求用户 |
| 场景词 | 使用场景+产品 | 差异化切入 |
| 问题词 | how/why/what/best for | 有购买意图的教育型词 |

### 3.3 标注优先级

| 优先级 | 标注标准 |
|--------|---------|
| A级 | 月搜索量>5000 + 与产品强相关 + 竞品转化词 |
| B级 | 月搜索量1000-5000 + 相关 |
| C级 | 月搜索量<1000 + 长尾/碎片词 |

---

## Step 4: 关键词库管理与广告绑定

### 4.1 词库管理（Excel/表格管理）

**关键词库字段规范：**
```
关键词 | 月搜索量 | 竞争度 | 分层(A/B/C) | 意图类型 | 来源ASIN | 广告状态 | 备注
```

**广告状态枚举：** 未投放 / 自动广告跑出 / 手动精准已投 / 否定 / 待验证

### 4.2 广告架构绑定（阿波罗广告架构）

| 关键词类型 | 绑定广告类型 | 匹配方式 | 出价策略 |
|-----------|------------|---------|---------|
| 核心大词 | 自动广告 + 手动广泛 | 广泛/词组 | 建议竞价×1.0-1.3 |
| 二级词 | 手动精准 | 精准/词组 | 建议竞价×0.8-1.0 |
| 长尾词 | 手动精准 | 精准 | 建议竞价×0.6-0.8 |
| 品牌竞品词 | ASIN定投 | 竞品ASIN | 根据转化调bid |
| 属性/场景词 | 手动词组 | 词组 | 建议竞价×0.6-1.0 |

### 4.3 持续循环优化（每周/月）

**周循环：**
1. 导出广告搜索词报告（SP）
2. 从"自动广告跑出词"中提取新词 → 分层 → 补入词库
3. 识别高点击低转化词 → 加入否定词库

**月循环：**
1. 重新对核心竞品做关键词反查（类目词可能有季节性变化）
2. 更新搜索量数据
3. 淘汰搜索量骤降的词
4. 新增趋势词（参考Google Trends）

## 关键决策点

| 决策点 | 判断标准 | 结果 |
|--------|---------|------|
| 竞品池选择 | 10-15个ASIN覆盖核心+新品+品牌 | 完成/补充 |
| 关键词分层 | 按搜索量/意图分类 | A/B/C级标注 |
| 入库筛选 | 删除明显不相关词 | 通过/淘汰 |
| 广告绑定 | 按分层确定投放策略 | 入自动/手动/ASIN定投 |

## 注意事项

1. **自动广告是找词工具，不是赚钱工具：** 自动广告跑出来的词质量高，但ACOS高，要及时把好词转移到手动精准
2. **否定词和关键词同等重要：** 否定词库建得好，能省30%+的无效花费
3. **词库按类目分文件：** 每个品类单独一个关键词库文件，不要混在一起
4. **月度更新不要全量替换：** 保留历史数据，新增/更新/淘汰分别标注，便于对比搜索量变化
5. **品牌竞品词用于防御：** 观察哪些品牌词跑到自家ASIN，如果自家品牌词被竞品抢，及时开品牌广告反击
