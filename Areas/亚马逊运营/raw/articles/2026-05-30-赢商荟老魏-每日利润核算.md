## 核心提炼（入库内容）

### 一、每日利润监控四维度

每天统计：
- **单量**：今日出单数量
- **销售额**：今日销售金额
- **广告支出**：今日广告花费
- **毛利润**：今日可核算毛利润

### 二、利润率公式（四层递减）

```
毛利率 = (售价 - 产品成本 - 头程物流成本) ÷ 售价
净利率 = 毛利率 - 广告费占比 - 退款占比 - 杂费占比
```

其中：
- 广告费占比 = 当月广告支出 ÷ 当月销售额
- 退款占比 = 当月退款金额 ÷ 当月销售额
- 杂费占比 = (仓储费 + 长期仓储费 + 店铺月租 + 提款佣金) ÷ 当月销售额

**判断标准：**
- 净利率为正 → 当前赚钱
- 净利率为负 → 当前亏损
- 净利率 > 15% → 合格

### 三、订单级费用核查SOP

**触发时机**：产品出第一单时立刻执行

1. 进入「付款 → 交易一览」
2. 找到该订单的明细
3. 核查亚马逊扣除的费用：FBA费 + 佣金比例是否正确
4. 若被多收 → **立即开CASE索赔**
5. 若正确 → 牢记，后续每月对比是否变化
6. 若中途又被多收 → 及时发现并开CASE要回

### 四、警示案例

- **错误做法**：广告占营收一半 + 长期仓储费多 + 亏本清货 = 必亏
- **根本原因**：平时不算账，凭感觉认为"低价产品一定赚钱"
- **教训**：15元拿货卖10美金≠一定赚钱，成本结构决定生死
