# Skill 系列 02：用 ABA 关键词报告，让 Agent 自动挖掘亚马逊细分市场

> 来源：跨境点AI Johnson，Skill系列第二篇
> 核心：从ABA Top Search Terms CSV出发，通过AI自动解析→聚类→验证→评分→输出Excel报告，挖掘亚马逊细分市场机会

## 核心思路

不是从产品出发，而是从**关键词数据反推细分市场**。

同样一个 dash camera，用户搜索的可能是不同需求：front and rear、parking mode、4K、night vision、truck/RV 等。从 ABA 热搜词入手，能看到更细的需求结构。

**ABA 能看到什么：** 用户在搜什么、点击给了谁、转化被谁拿走。
**ABA 不能看到什么：** 月搜索量、购买率、PPC竞价、供需比、搜索趋势。

## 完整流程（5个Phase）

### Phase 1：ABA CSV 解析
- 提取元数据（搜索词、报告周期）
- 提取正式数据（Search Term、Search Frequency Rank、Top Clicked Brand、Click Share、Conversion Share）
- 计算聚合指标：Top1/Top3点击份额、转化率（ABA推导，非后台CVR）
- 提取高频修饰词：去掉基础产品词和停用词后的修饰词（如 front/rear/4k/parking）
- 区分品牌词和非品牌词
- 输出三个列表：modifier_words、non_brand_keywords、top_asins

### Phase 2：AI语义聚类，识别细分市场
5个聚类维度：
- 产品类型（front and rear dash cam / mirror dash cam / mini dash cam）
- 目标人群（truck drivers / rideshare drivers / fleet / parents / commuters）
- 使用场景（parking mode / accident evidence / road trip / fleet monitoring）
- 功能特征（4K / night vision / WiFi / GPS / capacitor / loop recording）
- 车型/安装场景（car / truck / motorcycle / RV / rear view mirror）

聚类规则：
- 每个关键词只分配到一个细分
- 通用词归入最宽泛分类
- 品牌词单独标记，不计入分析
- 一个细分至少包含3个关键词
- 命名要清楚（如"Front and Rear Dash Camera - Parking Mode"）

每个细分的统计指标：
- 关键词数量、ABA最佳排名
- 平均Top1/Top3点击份额（判断垄断程度）
- 平均转化率（判断购买意图）
- 代表性关键词、ASIN、主要品牌

### Phase 3：卖家精灵 MCP 验证
**关键词级验证**（核心关键词 → keyword_research）：
- 月搜索量 searches、月购买量 purchases、购买率 purchaseRate
- 供需比 supplyDemandRatio、PPC竞价 bid、点击集中度 monopolyClickRate
- 平均售价 avgPrice、商品数 products、搜索增长率 searchGrowthRate

**ASIN级验证**（Top1代表ASIN → asin_detail）：
- 售价、评分、评论数、BSR、类目路径、上架时间、品牌

**可选深度验证**（最有潜力的2-3个细分）：
- keyword_miner：SPR、标题密度、广告竞品数
- google_trend：趋势验证（长期稳定/季节性/短期热词）
- traffic_keyword：ASIN实际流量词结构
- aba_research_weekly：周度趋势、异动词、持续增长词

### Phase 4：5维评分体系

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 市场规模 | 25% | 月搜索量>50k/ABA排名<10k→9-10分；20k-50k→7-8分；5k-20k→5-6分；1k-5k→3-4分；<1k→1-2分 |
| 转化潜力 | 20% | 购买率>5%+ABA转化率>70%→9-10分；3%-5%+50%-70%→7-8分；1.5%-3%+30%-50%→5-6分 |
| 竞争难度 | 25% | 点击集中度<30%+SPR<10→1-2分(低难度)；>80%→9-10分(高难度) |
| 成长性 | 15% | 搜索增长率>30%→9-10分；10%-30%→7-8分；0%-10%→5-6分；负增长→低分 |
| 利润空间 | 15% | 售价>$25+PPC<$1.5→9-10分；$15-25→7-8分；$10-15→5-6分；<$5→1-2分 |

**综合分 = 市场规模×0.25 + 转化潜力×0.20 + (10-竞争难度)×0.25 + 成长性×0.15 + 利润空间×0.15**

推荐等级：S级≥7.5（强烈推荐）→ A级6.0-7.4（值得进入）→ B级4.5-5.9（谨慎考虑）→ C级<4.5（不推荐）

### Phase 5：Excel报告（4个Sheet）
1. **细分市场总览**：按评分降序，含搜索量、购买率、PPC、供需比、竞品信息、推荐等级
2. **关键词明细**：每个关键词的ABA排名、点击份额、转化份额、所属细分
3. **ASIN排行**：从ABA反推的竞争ASIN频次排行（售价/评分/评论数）
4. **进入策略建议**：Top3细分各写一份策略（市场概况/售价区间/PPC/差异化方向/风险/回报周期）
