# Rufus 购物助手

## 什么是 Rufus

Rufus 是亚马逊 2024 年推出的 AI 购物助手，基于生成式 AI 运行，定位是「买家的智能购物伙伴」。月活已突破 **3 亿**，成为不可忽视的流量入口。

## 最新进展（2026年5月13日）

**Rufus 合并进 Alexa for Shopping**：亚马逊将 Rufus 整体并入 Alexa for Shopping，能力升级：

- 免 Echo / 免 Prime / 免 App，对全美购物者**免费开放**
- 核心变化：多了一个「AI 摘要 + 一键加购」的**答案位**层
- 详见：[[Alexa_for_Shopping与答案位]]

## 历史能力（已被 Alexa for Shopping 接管）

| 能力 | 说明 | 对卖家影响 |
|------|------|-----------|
| **意图预测** | 用户还没搜，主动推荐相关品类 | 品类卡位变得重要 |
| **问答购物** | 用户提问，Rufus 基于页面内容回答 | Listing QA 质量直接影响转化 |
| **比价分析** | 对比多个 ASIN 的参数/价格 | 差异化卖点必须清晰 |
| **场景推荐** | "for X scenario" 类型推荐 | 场景词覆盖决定曝光 |
| **浏览行为权重** | 用户在页面的浏览深度影响排名 | 需要优化停留时间和浏览路径 |

## 如何让 Alexa/Rufus 认识你的产品

### GEO 优化（Genuine Understanding Optimization）
核心目标：让 Alexa for Shopping 能准确理解你的产品属性。

**实操要点：**
1. **标题前 80 字符**：最关键，AI 摘要直接引用
2. **五点描述**：结构化属性 + 使用场景（避免形容词堆砌）
3. **A+ 内容**：加入使用场景图、对比图、人群图（A+ 对比表是 AI 摘要高频引用区）
4. **QA 内容**：植入高频场景问题（Alexa 会基于这些回答用户）
5. **评论内容**：包含真实使用场景描述

### 品牌广告 + Alexa/Rufus
建立品牌认知后，Alexa 会优先推荐：
1. 先用 DSP 展示广告 + SB 品牌广告建立品牌认知
2. 让 Alexa 建立品牌-品类关联
3. 当用户搜索品类词时，Alexa 优先推荐该品牌
