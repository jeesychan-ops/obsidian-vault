# AI时代Amazon高阶选品：算法友好型产品四维模型

## 核心认知转变

亚马逊正在从**"关键词搜索电商"**演化为**"基于AI的用户决策系统"**。

- 过去卖家研究：Keyword、Ranking、PPC、Review
- 未来高阶卖家研究：**Intent Graph、Semantic Structure、AI Recommendation、User Scenario、Behavioral Feedback**

平台关心两件事：
1. **Relevance**：商品和搜索词是否匹配
2. **Performance**：给你流量后平台是否能赚更多钱

---

## 推广难度公式

$$推广难度 \propto \frac{竞争强度 \times 流量成本}{产品效率差}$$

产品效率差 = CTR差 + CVR差 + Session Value差 + Review信任差 + Query Match差

Amazon持续计算：`E = CTR × CVR × AOV × Retention`

**真正难的不是竞争人数多，而是你能否在核心Query上打赢当前Top Results。**

---

## 六大推广难度维度

### 维度1：核心Query是否"固化"

air fryer、protein powder、gaming chair等词，用户已形成品牌偏好、Review信任和点击惯性，算法已形成稳定CTR/CVR模型。

判断标准：Top10品牌高度集中 + 评论极高 + 排名长期稳定 + Sponsored固定 → Query已成熟垄断，新品极难打破。

### 维度2：盈亏平衡CVR

$$BreakEvenCVR = \frac{CPC}{利润}$$

- CPC=$4，利润=$15 → CPA=$40，数学上无法盈利
- CPC=$2，利润=$20 → CVR必须>10%，若行业平均CVR=7%，天然难推

### 维度3：差异化程度（CTR维度）

差异化越弱 → CTR越低 → 推广越难。典型低差异化类目：数据线、手机壳、普通白牌产品。

### 维度4：Review信任（CVR放大器）

0 review / 5 review / 500 review 对应的CVR差异是**指数级**的。新品冷启动的真正难点是系统不信任你，高客单/功能型/安全类产品尤甚。

### 维度5：流量生态四阶段

| 阶段 | 特征 | 推广难度 |
|------|------|---------|
| 新品友好期 | 搜索量增长、竞争少、CPC低 | ⭐ 容易 |
| 成长期 | 品牌出现、广告变贵 | ⭐⭐ 中等 |
| 垄断期 | Top稳定、老品垄断、CPC极高 | ⭐⭐⭐⭐ 极难 |
| 衰退期 | 流量总盘子缩小 | ⭐⭐⭐ 结构难 |

### 维度6：流量来源分散度

依赖搜索流量的类目必须正面硬刚核心关键词；流量来自Related Products / SB视频 / 场景推荐 / 外部种草 / DSP的类目，绕开Query战争，推广难度反而低。

---

## 难推类目的三个特征

1. **高Query集中**：大量销量来自少数超级大词（creatine、air fryer、office chair）
2. **高历史行为固化**：用户形成品牌认知和排名依赖，新品难以改变点击行为
3. **广告变门票**：不开广告没流量，开广告不赚钱（最危险阶段）

---

## AI时代选品：从"低竞争"到"算法友好"

- **过去的选品逻辑**：找低竞争、找蓝海、找低Review
- **未来的选品逻辑**：找**算法友好型产品**

即使竞争低，没有语义定位、没有场景、没有人群、没有情绪需求 → 系统依然无法推荐你。

---

## 算法友好型产品四维模型

### ✅ 特征一：强场景
- 错误：`desk lamp`
- 正确：`desk lamp for ADHD study`

场景越明确，系统越容易理解用户意图。

### ✅ 特征二：强人群
`for gamers` / `for moms` / `for small apartments` / `for pet anxiety`

Amazon正在变成"人群推荐系统"，强人群定位直接决定AI能否精准推送。

### ✅ 特征三：强问题解决能力
未来AI搜索的核心逻辑：**问题 → 解决方案**。解决问题越明确，越容易进入AI Recommendation。

### ✅ 特征四：强语义标签
用户反复用相同语言描述的词会成为AI语义入口：`back pain` / `calming` / `ergonomic` / `portable` / `sensory` / `space saving`

**只有"商品属性"而无"语义属性"的产品未来会越来越难**（普通数据线、手机壳、水杯）。

---

## 未来竞争维度转变

| 过去 | 未来 |
|------|------|
| Keyword Competition | Intent Competition |
| Ranking Competition | Semantic Competition |
| | **AI Recommendation Competition** |

核心变化：不再只是"你卖什么"，而是 **"AI认为你解决什么问题"**。
