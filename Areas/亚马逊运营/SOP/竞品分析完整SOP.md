# 竞品分析完整SOP

## 第零步：准备工作 — 数据调取链路（AI辅助模式）

> 来源：LinkFox Agent 竞品分析工具链（2026-05-28）
> 说明：竞品分析80%的工作量在于数据调取，AI辅助可将时间从"2小时"压缩到"10分钟"

### 三段式数据调取流程

```
┌──────────────────────────────────────────────────────┐
│  段1: 数据调取  →  段2: 节奏还原  →  段3: 文案生成  │
│  (工具抓取)          (Keepa分析)        (AI优化)     │
└──────────────────────────────────────────────────────┘
```

| 阶段 | 工具 | 核心动作 | 输出 |
|------|------|---------|------|
| **段1：数据调取** | 亚马逊搜索模拟器 + 图片相似度分析 | 按产品类型/材质/价格分组排列竞品 | 分组后的竞品清单 |
| **段2：节奏还原** | SIF关键词 + Keepa | 查竞品关键词 + 推广节奏 + 产品变化时间线 | 运营节奏表 + 关键词价值分析 |
| **段3：文案生成** | COSMA算法/SIF + AI | 优化五点描述 + 标注埋词位置 | 可直接使用的文案 |

### 关键词价值分析标准

| 指标 | 含义 | 用途 |
|------|------|------|
| 搜索量 | 月均搜索量 | 判断流量规模 |
| 流量占比 | 该词占竞品总流量% | 识别核心词 vs 边缘词 |
| 自然/广告分布 | 自然位 vs 广告位占比 | 判断竞品广告投入程度 |

### 竞品分组逻辑（AI辅助自动完成）

- 竹制 / 塑料 / 手摇 / 弓形（按材质/形态分组）
- 或按价格带分组：<$20 低端 / $20-40 中端 / >$40 高端
- **目的**：同组竞品才是真正对手，便于差异化定位

## 核心框架：7步分析法

> 来源：拆解100个对手后，我总结了这套竞品分析方法
> 适用场景：日常竞品监控 / 新品调研 / 爆款拆解

### 第一步：建竞品池（分层管理）

| 层级 | 定义 | 分析目标 |
|------|------|---------|
| 头部竞品 | 类目BSR前3-5名 | 学主流打法、定天花板 |
| 同级竞品 | 价格/体量接近但销量更好 | 找流量来源、抢转化 |
| 黑马竞品 | 上架<6个月、BSR快速攀升 | 学切入方式、差异化定位 |

**筛选标准：**
- 月销量 ≥ 500单（排除短期冲量产品）
- 评论增长趋势持续（非一次性爆量）
- 价格带与自身产品接近

### 第二步：市场值不值得做

创建竞品分析表，快速判断市场格局：

- **大盘**：价格带是否集中？首页评论门槛？品牌垄断程度？
- **基本信息**：品牌 / ASIN / 上架时间 / BSR排名
- **销售表现**：月销量 / 价格区间 / Review数量及趋势 / 星级

### 第三步：还原竞品运营节奏

通过运营时光机（如 Sorftime Keepa）剖析竞品从上架开始的运营时间线：

- **评论增长**：哪个时间段上了多少评论？
- **页面优化**：图片、标题、价格每次调整记录
- **促销动作**：是否叠加Coupon / 秒杀？与BSR变动关系？

典型爆款节奏规律：
> 第7天冲秒杀 → 第10天集中上评论 → 第14天改主图 → 第20天视频广告冲排名

### 第四步：解剖竞品广告逻辑

搞清楚对手的钱花在哪、怎么花的：

1. 用工具（如Sif）查广告架构：开了多少广告活动？SP/SB/SBV比例？
2. 分析每个广告活动的：投放词根 / 匹配类型 / 投放变体
3. 通过"词根+否定词"反推匹配类型（广泛/词组/精准）
4. 判断广告与自然词的协同关系

**广告结构判断标准：**
- 核心词同时出现在自然排名 → 自然词在支撑广告权重
- 评论集中增长期 + 广告放量 → 配合促销
- 是否由其他变体承接广告 → 流量分摊策略

### 第五步：精读页面，拆解转化率

| 模块 | 关键问题 |
|------|---------|
| 标题 | 核心词/长尾词/卖点如何融合？ |
| 主图视频 | 第一张图是否一眼讲清产品？辅图有尺寸图/场景图/对比图？ |
| 五点/A+ | 文案有没有提前回答用户犹豫的问题？ |
| 差评预判 | 页面是否已经解决了差评中的常见投诉？ |

好的链接 = 在图片和文案里提前回答用户会犹豫的问题。

### 第六步：深挖评论，找到优化机会

**重点看3星和4星评论**，把高频问题分类：

- **结构问题**：尺寸不准、易坏 → 优化即竞争优势
- **体验问题**：难安装、设计不合理 → 改进包装或微调产品
- **包装问题**：破损 → 加强保护层
- **预期落差**：和图不符 → 文案别夸大、主图别过度渲染

整理成"竞品漏洞清单"，直接指导选品和文案。

### 第七步：反查关键词，锁定流量密码

- **看流量结构**：多少来自自然搜索？多少来自广告？
- **看广告构成**：SP / 品牌广告 / 视频广告各占多少？
- **反查出单词**：高流量占比 + 稳定出单 = 核心词，重点布局；高流量但转化差 = 验证过的"坑"，投放时绕开

## 可复用判断结构

> 来源：亚马逊竞品分析：建立一套可复用的判断结构（李李）

### 竞品分层与判断用途

```
类目层 → 理解市场整体结构（产品形态/平台偏好表达方式）
搜索层 → 找直接竞争对手（反复出现 + 增长节奏接近）
黑马层 → 拆解推广节奏和阶段性策略
```

### 差异化是否成立的验证标准

| 验证方式 | 作用 |
|---------|------|
| Review 高频描述 | 判断竞品主要传递的价值方向（价格/外观/专业性/安全性/便利性） |
| 差评是否提及该差异 | 无法在用户反馈中验证的差异，不作主要决策依据 |
| QA 是否围绕展开 | 有QA说明用户真正关心这个问题 |

### 差异点的执行成本分级

| 差异类型 | 执行成本 | 优先级 |
|---------|---------|-------|
| 结构或功能差异 | 高 | 低优先级 |
| 配件或组合差异 | 中 | 中优先级 |
| 图片与表达差异 | 低 | **高优先级** |

> 优先关注执行成本较低、验证周期较短的调整方向。

### 常见误判场景

1. 只看到销量上涨，忽略是否叠加了促销和广告放量
2. 只看到价格变化，没有同步查看流量结构
3. 只拆广告动作，却没有结合自然词变化

## 竞品分析决策树（阿澄版）

> 来源：竞品分析怎么做？6年亚马逊老运营教你一键拆解爆款打法

```
值得盯的竞品？
├── 同级竞品（同定价/体量，但销量更好）→ 学它优于你的地方
└── 黑马竞品（3-6个月上架，BSR快速上升）→ 学流量入口+打法+差异化

分析维度：
1. 流量来源 → 关键词（核心词+长尾词）+ 站内/站外入口
2. 产品痛点 → 差评找机会，好评找精准卖点词
3. 打法成本 → 算清是正常推广还是亏本冲量
```

### 差评/好评的用途区分

| 类型 | 内容来源 | 用途 |
|------|---------|------|
| 1-3星差评 | 功能短板/设计/材质/安装问题 | 产品改进方向 + 文案痛点 |
| 4-5星好评 | 买家核心满意点 | 标题/五点精准流量词来源 |

## 爆款运营四阶段节奏（案例印证）

> 来源：不急不躁却从0做到2000+月销，大促后我们被它打动了

季节性产品（8月→12月旺季）完整推广节奏：

| 阶段 | 时间 | 核心动作 | 关键数据指标 |
|------|------|---------|------------|
| **测试期** | 8-10月 | 泛化广告测收录 + Vine打口碑 + 价格测试 | BSR稳步上升 |
| **成长期** | 11月 | 聚焦2个核心精准词，抢占关键词权重 | 核心词自然排名前10 |
| **爆发期** | 12月 | 拓展场景词（如Gift类）+ 逐步涨价54% | BSR前5 + 利润最大化 |
| **衰落期** | 12月底 | 降价清仓 + 广告减少 | 库存健康周转 |

**广告演进规律：**
- 阶段1：自动泛化广告（测收录）+ 词根匹配广告（测核心词）
- 阶段2：精准产品词卡位
- 阶段3：场景词拓展 + 利润收割
- 阶段4：降频收尾
