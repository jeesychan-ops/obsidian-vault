# ABA关键词细分市场挖掘法

> **核心思路：** 不是从产品出发，而是从ABA（Amazon Brand Analytics）Top Search Terms CSV出发，用AI自动解析关键词→语义聚类识别细分市场→卖家精灵MCP验证→5维评分→输出Excel报告。
>
> 与 [[趋势选品雷达]]（Google Trends出发）和 [[率化选品SOP]]（7维评分）互补：本框架解决的是"在大品类中找到细分切入口的结构化方法"。

---

## 一、5段式完整流程

### Phase 1：ABA CSV 解析
- 提取元数据（搜索词、报告周期）
- 正式数据（Search Term、Search Frequency Rank、Top Clicked Brand/ASIN、Click Share、Conversion Share）
- 计算聚合指标：Top1/Top3点击份额、Top3转化率
- 提取高频修饰词（去基础词+停用词后的修饰词）
- 区分品牌词和非品牌词
- 输出：modifier_words / non_brand_keywords / top_asins

### Phase 2：AI语义聚类
按5个维度聚类——**产品类型** / **目标人群** / **使用场景** / **功能特征** / **车型/安装场景**

聚类规则：
- 每个关键词只分配到一个细分
- 通用词归入最宽泛分类
- 品牌词单独标记不计入
- 细分至少包含3个关键词才有统计意义
- 命名清晰（如"Front and Rear Dash Camera - Parking Mode"）

每个细分统计：关键词数量、ABA最佳排名、平均Top1/Top3点击份额、平均转化率、代表ASIN/品牌

### Phase 3：卖家精灵MCP验证
- **关键词级** → keyword_research：月搜索量、购买率、PPC竞价、供需比、点击集中度
- **ASIN级** → asin_detail：售价、评分、评论数、BSR、上架时间
- **深度验证** → keyword_miner / google_trend / traffic_keyword：SPR、趋势、流量结构

### Phase 4：5维评分体系

| 维度 | 权重 | 关键判断依据 |
|------|------|-------------|
| 市场规模 | 25% | 月搜索量 + ABA排名 |
| 转化潜力 | 20% | 购买率 + ABA转化率 |
| 竞争难度 | 25% | 点击集中度 + SPR + 供需比（注意：维度分越高=越难） |
| 成长性 | 15% | 搜索增长率 → 长期稳定/季节性/短期热词 |
| 利润空间 | 15% | 售价 + PPC竞价 |

**综合分公式：** `市场规模×0.25 + 转化潜力×0.20 + (10-竞争难度)×0.25 + 成长性×0.15 + 利润空间×0.15`

| 综合分 | 等级 | 含义 |
|-------|------|------|
| ≥7.5 | S级 | 强烈推荐，市场大+转化好+竞争可控 |
| 6.0-7.4 | A级 | 值得进入，综合条件较好 |
| 4.5-5.9 | B级 | 谨慎考虑，存在一定风险 |
| <4.5 | C级 | 不推荐，竞争激烈或市场太小 |

### Phase 5：Excel报告
4个Sheet：
1. **细分市场总览** — 按评分降序，机会地图
2. **关键词明细** — 原始颗粒度，用于广告+Listing延伸
3. **ASIN排行** — 竞争ASIN频次排行
4. **进入策略建议** — Top3细分各写策略（定价/PPC/差异化/风险）

---

## 二、关键判断指标

**最重要的两个诊断指标：**
1. **Top1/Top3点击份额** — 判断市场垄断程度。搜索量大但Top3集中度高 → 看起来大但新产品难拿流量
2. **ABA转化率** — 非后台CVR，而是ABA转化份额/点击份额推导的相对指标，判断购买意图强弱

**核心洞察：** 如果一个细分搜索量中等、Top3点击份额不高、转化率不错 → 最适合中小卖家切入的机会。

## 四、使用限制

1. 依赖ABA数据质量（关键词<20个时建议提示用户扩充）
2. ABA只有搜索/点击/转化份额，不含利润模型 → 最终决策需补供应链/物流/认证/专利
3. AI聚类需人工复盘优化命名和归类规则
4. 卖家精灵MCP可能失败 → 记录错误，用ABA数据单独评估，标注"未验证"
