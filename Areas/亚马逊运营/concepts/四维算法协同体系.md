# 四维算法协同体系

## 一句话总结

> - [[A9A10算法]] 解决「搜得到」和「排得高」
> - [[COSMO算法]] 解决「搜的人是不是对的人」
> - [[Alexa_for_Shopping与答案位]] 解决「没搜的人怎么被种草」+「AI 代理购买决策」
> - [[A9搜索意图六分法]] 补充：Amazon 主动预测用户下一步搜索，搜索路径决定推荐

四个算法维度同时在跑，广告策略必须同时照顾。

## 四维算法详解

### 维度一：A9/A10（传统搜索）
| 维度 | 内容 |
|------|------|
| 核心逻辑 | 关键词相关性 + 转化率权重 |
| 流量来源 | 搜索结果自然排名 + 广告排名 |
| 优化要点 | 关键词埋词、点击率、转化率、账户权重 |
| 权重因素 | 转化率、销量、评论、FBA库存 |

### 维度二：A10（站外权重升级）
| 维度 | 内容 |
|------|------|
| 核心逻辑 | 增加站外信号权重 |
| 新增权重 | 社媒提及、独立站引流、品牌搜索量 |
| 优化要点 | 站外引流 + Amazon Attribution 追踪 |
| 效果 | 站外成交 + 站内 SD 再营销 = 自然排名飙升 |

> 案例：家居品牌 TikTok 视频爆了（500 万播放），引导到亚马逊旗舰店，站外成交 $12,000，站内 SD 再营销 ACOS 仅 8%，品牌词搜索量一周涨了 340%，自然排名冲到前 5。

### 维度三：COSMO（意图匹配）
| 维度 | 内容 |
|------|------|
| 核心逻辑 | 从关键词匹配升级为用户意图理解 |
| 流量特征 | 场景词、人群词、个性化推荐 |
| 优化要点 | 语义完整度 > 关键词密度 |
| 核心问题 | 这个词背后的真实需求是什么？ |

### 维度四：Alexa for Shopping（AI 主动推荐，Rufus 已合并）

**2026-05-13 更新**：Rufus 已整体合并进 Alexa for Shopping，能力升级为「AI 摘要 + 一键加购」的答案位层，详见 [[Alexa_for_Shopping与答案位]]。

| 维度 | 内容 |
|------|------|
| 核心逻辑 | AI 主动推荐，用户还没搜就推给你；新增 Buy for Me 代理购买模式 |
| 流量特征 | 品类卡位、问答购物、比价分析、答案位、AI 摘要直接说服 |
| 优化要点 | QA 内容质量、场景词覆盖、GEO 优化、组合关系表达、外部内容生态 |
| 核心问题 | 如何让 Alexa 准确理解并推荐我的产品？ |

新增核心策略（RPO-推理路径优化）：
- 产品从「product description」重写为「task solution」
- Reddit/Medium 布局 AI 外部数据源
- Q&A 场景化 + Review 场景关键词植入
- 组合关系表达（"works with X"）大幅提升推荐概率

## 意图分层投放（广告 × 算法协同）

基于四维算法，关键词可分为四种意图，对应不同广告策略：

| 意图类型 | 搜索词示例 | 广告类型 | 竞价策略 |
|---------|---------|---------|---------|
| 信息型 | how to, best, review | 内容种草，低竞价测试 | 低竞价跑数据 |
| 对比型 | vs, compare, alternative | 商品投放竞品 ASIN | 中等竞价 |
| 场景型 | for, with, when | COSMO 长尾词重点布局 | 中等竞价 |
| 购买型 | buy, deal, discount | 精准匹配大词 | 高竞价守位 |

## A9/A10 vs COSMO 的新旧打法对比

| 维度 | 旧打法 | 新打法（Alexa/COSMO 时代） |
|------|--------|--------------------------|
| 关键词策略 | 大词抢首页，长尾词铺量 | 意图分层投放 + 搜索路径地图 |
| 埋词逻辑 | 关键词密度 | 语义完整度（4 个问题完整回答） |
| 匹配方式 | 自动广告跑词 | 自动 + 场景词精准 |
| 排名逻辑 | 转化率高 = 排名高 | 用户画像匹配度 = 排名高 |
| 新维度 | - | AI 推理链路（RPO）= 进入 AI 答案位 |

## 全渠道组合（广告协同）

单一 SP 广告 ROI 有天花板，**SP+SD+SB+DSP 组合才能实现指数级增长**：

| 广告类型 | 覆盖 Funnel 阶段 | 核心目标 |
|---------|---------------|---------|
| SP（商品推广） | 中下层（已有购买意图） | 精准收割 |
| SD（展示型广告） | 中上层 + 再营销 | 拉新 + 激活 |
| SB（品牌推广） | 上层（品牌认知） | 建立品牌 |
| DSP | 全层（站外） | 站外引流反哺站内 |

## 与其他概念的关系

- 是 [[万词打法]] 的算法基础：万词打法需要同时针对四个算法维度进行优化
- 指导 [[广告策略]] 的精细化：不同算法维度对应不同的广告类型和关键词策略
- 与 [[站内流量]] 直接关联：四维算法共同决定了自然搜索流量的分配
- 最终通过 [[A9A10算法]] 的权重机制实现「广告带自然、自然带爆单」的飞轮效应
