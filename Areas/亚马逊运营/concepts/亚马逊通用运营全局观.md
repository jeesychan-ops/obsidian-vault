# 亚马逊通用运营全局观

## 核心框架

亚马逊运营的本质是一个**飞轮**：曝光→点击→转化→评价→排名→曝光。每个环节相互咬合，飞轮转起来才能真正爆款。

> **爆款公式** = 选品（40%）+ Listing质量（30%）+ 广告效率（20%）+ 评价积累（10%）

---

## 运营漏斗

```
         曝光
           │
     ┌─────▼─────┐
     │   点击     │ ──── 流量获取
     └─────┬─────┘
           │
     ┌─────▼─────┐
     │   转化     │ ──── 转化优化
     └─────┬─────┘
           │
     ┌─────▼─────┐
     │   复购    │ ──── 用户运营
     └─────┬─────┘
           │
           ▼
         利润
```

| 环节 | 核心指标 | 优化方向 |
|------|---------|---------|
| 曝光 | Listing 被展示次数 | 关键词收录、BSR权重、类目节点 |
| 点击 | CTR = 点击/曝光 | 主图、价格、评分、评分数量 |
| 转化 | CVR = 订单/点击 | Listing质量、评论、价格、A+ |
| 复购 | 重复购买占比 | 产品质量、客服、包装 |
| 利润 | 净利润/销售额 | 成本控制、定价策略、仓储优化 |

## 运营阶段诊断

每个阶段的特征、核心任务和打法完全不同。**不知道自己处于哪个阶段，是大多数运营失误的根源。**

### 初创期（0~60天）

**特征：**
- Listing 新上架，评论少或无评论
- 关键词排名靠后，曝光有限
- 转化率不稳定

**核心任务：**
1. Listing 优化完成度（标题/Bullets/A+/QA/属性）
2. 评论积累（Vine计划、催评策略）
3. 基础关键词收录
4. 广告冷启动（只开1个自动广告，简单竞价）

**参考：[[老魏极简化广告策略]]**

---

### 成长期（60~180天）

**特征：**
- 稳定出单，部分关键词有排名
- 评论逐步增加，但评分可能波动
- 利润压力开始显现

**核心任务：**
1. 关键词层级提升（长尾→中等→核心大词）
2. ACOS 优化（广告结构精细化）
3. 自然流量占比提升（广告带动自然排名）
4. 差异化卖点提炼

**核心诊断：** "这些词，能不能支撑你继续放量？"

---

### 成熟期（180天+）

**特征：**
- 销量相对稳定，BSR排名靠前
- 品牌在细分品类有一定认知
- 同质化竞争加剧

**核心任务：**
1. 品牌建设（A+、品牌旗舰店、品牌保护）
2. 产品线扩展（变体、子ASIN、关联产品）
3. 多渠道布局（站外、其它平台）
4. 供应链优化（成本谈判、备货效率）

---

### 突破期（任何阶段可能触发）

**特征：**
- 增长遇到瓶颈
- 竞争加剧，ACOS上升
- 成本上升，利润压缩

**核心任务：**
1. 差异化升级（材质/功能/包装/服务）
2. 供应链成本优化
3. 精细化运营（广告acos管控、库存周转）
4. 新品补充（第二条增长曲线）

## AI 全链路应用

AI 不是单一工具，而是贯穿选品→Listing→广告→客服→数据分析的全链路能力。

```
选品 ──► Listing ──► 广告 ──► 客服 ──► 数据分析
  │        │          │         │          │
  ▼        ▼          ▼         ▼          ▼
AI辅助   AI生成    AI优化   AI回复   AI分析
市场分析 标题/五点 关键词策略 自动回复 销售预测
竞品调研 A+内容    出价优化  差评处理  利润分析
趋势预测 QA生成   预算分配  FAQ生成   竞品监控
```

**首个全链路案例（2026-04-27）：新品14天从0到日销100单**

## 核心指标体系

| 指标 | 定义 | 健康区间 | 优化方向 |
|------|------|---------|---------|
| ACOS | 广告花费/广告销售额 | 新品<30%，稳定期<20% | 降bid/提转化/优化关键词 |
| TACOS | 总花费（含自然）/总销售额 | <15%优秀 | 自然流量占比提升 |
| 转化率 | 订单数/点击数 | 类目平均的1.5x+ | 优化Listing + 评论 |
| BSR排名 | 类目畅销榜排名 | Top 100冲击中 | 销量稳定增长 |
| 评论分数 | 星评分 | 4.3+安全区 | 催好评+差评处理 |
| 库存周转 | IPI关联 | 尽量避免冗余 | 备货节奏管理 |
