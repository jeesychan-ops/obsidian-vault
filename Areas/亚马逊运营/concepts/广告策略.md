# 亚马逊广告策略体系

## 广告架构三层模型

### 第一层：自动广告（ASIN反查 + 关键词发现）
- 目的：让算法找到产品的目标人群
- 关键词来源：系统推荐词、竞品 ASIN 报告
- 适合新品期，低竞价跑数据

### 第二层：手动精准关键词广告
- 目的：锁定高转化的明确搜索词
- 策略：广泛 + 词组 + 精准三档测试
- 进阶：[[SOP-广告架构搭建流程]]

### 第三层：品牌广告 / DSP
- 目的：拉新和保护品牌词流量
- 适合已有一定销售基础的成熟产品

### 第四层（2026-06 新增）：商品投放（关联流量入口）
- **核心理念转变：抢占竞品详情页，而不只是搜索页**
- 类型：SP ASIN定投 + SD商品投放 + SB商品投放 + SBV视频广告
- 重点投放：BSR头部竞品 / 高流量新品 / 互补产品 / 高转化ASIN
- 详见 [[关联流量布局：Product Graph 打法]]

## 关键词竞争度决策框架

**核心问题不是"怎么分析竞争度"，而是该不该往中高竞争词走：**

| 阶段 | 竞争度 | 策略 |
|------|--------|------|
| 新品冷启动 | 低竞争词优先 | 稳住ACOS，积累数据 |
| 成长期 | 逐步进入中竞争 | 监测ACOS变化，及时调整出价 |
| 成熟期 | 中高竞争词 | 增长优先，接受略高ACOS |

## 万词打法（高阶）

四维协同：A9 + A10 + COSMO + Rufus

- A9/A10：算法排名权重优化
- COSMO：对话式搜索优化（用户真实意图）
- Rufus：买前浏览行为权重
- 万词：海量关键词覆盖，ACOS压到20%以内

详见：知识库/亚马逊广告高阶打法：A9+A10+COSMO+Rufus 四维协同

## ACOS 控制的核心逻辑

1. 关键词精准匹配，提高相关性
2. 落地页优化（Listing质量）提高转化率
3. 评论数量和质量提升 Social Proof
4. 合理竞价：不追第一，稳守前三页即可

## 与其他概念的关系

- 广告是 [[站内流量]] 的核心付费来源
- 广告效果依赖 [[Listing优化]] 的落地页质量
- 广告数据反过来指导 [[选品三维度法]] 的决策（哪个词好卖→哪个品类有机会）
- 广告表现与 [[A9A10算法]] 紧密相关：高质量广告 → 高转化 → 算法给更多自然流量
