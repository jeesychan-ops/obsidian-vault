# 亚马逊财务与利润分析

## 利润测算框架

### 收入端
- 售价 × 订单数量
- 促销/ Deals 期间的价格调整

### 成本端
| 成本项 | 说明 |
|--------|------|
| 产品成本 | 采购成本 |
| 头程运费 | 国内到 FBA 仓库 |
| 平台佣金 | 约 15%（类目不同） |
| FBA 仓储费 | 月度仓储 + 长期仓储 |
| 物流成本 | 退货处理费 |
| 广告费 | ACOS × 销售额 |
| 退货率 | 类目均值 5-15% |

### 净利润公式
```
净利润 = 售价 × 订单数 - 产品成本 - 头程 - 平台佣金 - 仓储费 - 广告费 - 退货损失
```

## 毛利率判断标准（参考 [[选品三维度法]]）

- 毛利率低于 **30%** 不碰
- 扣除广告 + 仓储后净利率 > **15%** 才算合格

## 盈亏平衡分析

- 新品期：广告费占比高，净利率低是正常的（投资期）
- 成长期：随自然流量占比提升，广告费占比下降，利润改善
- 成熟期：低 ACOS + 高自然流量 = 最佳利润结构

## 与其他概念的关系

- 财务测算决定了 [[选品三维度法]] 的可行性——再好的产品，利润算不过来也不值得做
- [[广告策略]] 的 ACOS 目标设定要以利润率为上限
- 利润分析指导 [[站内流量]] 的投入决策——自然流量占比高的阶段可适当降低广告费

## 新增：日度利润监控（老魏《增长飞轮2》）

> 来源：赢商荟老魏《创业者必须每天关注经营数字，每天核算利润表》

### 每日必盯四维度

每天统计每个账号/产品：
- **单量**：今日出单数量
- **销售额**：今日销售金额
- **广告支出**：今日广告花费
- **毛利润**：今日可核算毛利润

### 利润率四层递减公式

```
毛利率 = (售价 - 产品成本 - 头程物流成本) ÷ 售价
净利率 = 毛利率 - 广告费占比 - 退款占比 - 杂费占比
```

各层占比计算：
| 指标 | 公式 |
|------|------|
| 广告费占比 | 当月广告支出 ÷ 当月销售额 |
| 退款占比 | 当月退款金额 ÷ 当月销售额 |
| 杂费占比 | (仓储费 + 长期仓储费 + 店铺月租 + 提款佣金) ÷ 当月销售额 |

**判断：净利率为正=赚钱，净利率为负=亏损，>15%=合格**

### 订单级费用核查SOP

**触发时机：产品出第一单时立刻执行**

1. 进入「付款 → 交易一览」
2. 找到该订单明细，拆出 FBA 费 + 佣金比例
3. **若被多收 → 立即开CASE索赔**
4. 若正确 → 牢记，后续每月对比是否变化
5. 中途又被多收 → 及时发现并开CASE要回

### 常见亏损陷阱（老魏警示案例）

- ❌ 广告占营收一半 + 长期仓储费高 + 亏本清货 = 必亏
- ❌ "15元拿货卖10美金一定赚钱" → 成本结构不看清楚就是赌
- ✅ 平时不算账 → 年底盘点巨亏 → 为时已晚
