# 亚马逊财务与利润分析

## 利润测算框架

### 收入端
- 售价 × 订单数量
- 促销/ Deals 期间的价格调整

### 成本端
| 成本项 | 说明 |
|--------|------|
| 产品成本 | 采购成本 |
| 头程运费 | 国内到 FBA 仓库 |
| 平台佣金 | 约 15%（类目不同） |
| FBA 仓储费 | 月度仓储 + 长期仓储 |
| 物流成本 | 退货处理费 |
| 广告费 | ACOS × 销售额 |
| 退货率 | 类目均值 5-15% |

### 净利润公式
```
净利润 = 售价 × 订单数 - 产品成本 - 头程 - 平台佣金 - 仓储费 - 广告费 - 退货损失
```

## 毛利率判断标准（参考 [[选品三维度法]]）

- 毛利率低于 **30%** 不碰
- 扣除广告 + 仓储后净利率 > **15%** 才算合格

## 盈亏平衡分析

- 新品期：广告费占比高，净利率低是正常的（投资期）
- 成长期：随自然流量占比提升，广告费占比下降，利润改善
- 成熟期：低 ACOS + 高自然流量 = 最佳利润结构

## 与其他概念的关系

- 财务测算决定了 [[选品三维度法]] 的可行性——再好的产品，利润算不过来也不值得做
- [[广告策略]] 的 ACOS 目标设定要以利润率为上限
- 利润分析指导 [[站内流量]] 的投入决策——自然流量占比高的阶段可适当降低广告费
